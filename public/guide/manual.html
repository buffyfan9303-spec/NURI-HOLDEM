01회원가입 · 본인인증

  1. 우상단 로그인 → 업주 가입 탭 선택(일반 가입과 구분됩니다).
  2. 이메일·비밀번호·이름을 입력하고 약관에 동의하면 가입 완료.
  3. 프로필에서 본인인증을 진행하세요(휴대폰 인증). 만 19세 이상만 가능합니다.
✅ 본인인증을 완료해야 대회 예약·글쓰기·장터 등록 등 핵심 기능이 열립니다. 사장님 본인도 인증해 두는 것을 권장합니다.
⚠ 업주 가입 후에는 운영자 승인을 기다리는 상태가 됩니다 — 다음 단계(매장 개설)를 먼저 진행하세요.

02매장 개설 · 승인

  1. 하단 탭 내 매장 진입 → 내 매장 만들기 화면이 나옵니다.
  2. 대표 이미지(선택) · 매장명* · 주소* · 전화* · 지역 · 카카오톡 링크 · 영업시간 · 소개를 입력하고 생성.
  3. 운영자 검토 후 승인되면 알림이 오고, 매장이 일정탐색·커뮤니티에 공개됩니다.

인증 매장이 되면

  • 포스터가 승인 없이 즉시 게시되고 매장 목록 상단에 인증 배지와 함께 우선 노출됩니다.
  • 순위 입력의 지류 양식 인쇄 기능이 열립니다. (인증은 운영자가 검토 후 부여)

03내 매장 메뉴 지도

모바일은 상단 현재 메뉴를 탭하면 전체 메뉴가 펼쳐지고, PC는 좌측 사이드바입니다. ★를 눌러 자주 쓰는 메뉴를 상단에 고정할 수 있어요(최대 5개).

04매장 꾸미기

  • 내 매장 › 매장 꾸미기 — 배경/사진 여러 장, 소개글, 매장 페이지의 탭 구성·순서를 편집합니다.
  • 나만의 주소(슬러그)를 설정하면 nuriholdem.com/?v=내주소 형태로 공유하기 좋아집니다.
  • 매장 페이지의 정보 편집에서 주소·전화·영업시간을 언제든 수정할 수 있습니다.
✅ 사진과 소개가 채워진 매장이 팔로우·예약 전환이 눈에 띄게 좋습니다. 개설 직후 5분만 투자하세요.

05게임 프리셋

매주 반복하는 대회는 프리셋으로 저장해 두면 포스터·장부·클락에서 재사용됩니다.

  • 저장 항목: 제목·게임 종류·바인·스타팅/리바인/애드온 스택·GTD(또는 엔트리%)·듀레이션·블라인드 구조(레벨별 SB/BB/앤티/시간)·메모.
  • 블라인드 편집기에서 "15레벨까지 25분, 16부터 15분" 같은 일괄 듀레이션과 자동 생성이 가능합니다.
  • 클락 설정의 게임 프리셋에서 블라인드 불러오기로 바로 적용됩니다.

06포스터 등록 — 대회를 알리고 예약 받기

  1. 내 매장 › 포스터·예약 › + 새 게임
  2. 포스터 이미지 업로드 + 게임 이름* · 날짜* · 스타트 시간* · 바이인* · 상금(GTD/프라이즈%)* · 레지마감(레벨 또는 시간)* · 지역* · 결제 수단* 입력.
  3. 반복 등록(매주 같은 요일 4/8/12주)과 지난 포스터 불러오기로 재등록이 빨라집니다.
  4. 등록하면 운영자 승인 후 일정탐색 메인에 노출되고 예약이 열립니다(인증 매장은 즉시 게시).
✅ 등록 즉시 팔로워에게 새 포스터 알림이 자동 발송되고, 대회 전날 리마인더도 자동으로 나갑니다.
⚠ 승인 전에는 목록에 보이지 않습니다. 승인 요청은 운영자에게 자동 알림되니 기다리시면 됩니다.

부스트(상단 고정)

내 포스터를 일정탐색 맨 위에 N일 고정하고 TOP 배지를 답니다 — 대시보드의 부스트 카드에서 문의 방법을 확인하세요.

07예약 관리

  • 포스터·예약에서 게임을 누르면 예약 리스트가 펼쳐집니다 — 예약명·날짜·시간은 해당 매장 업주/직원만 볼 수 있어요.
  • 게임 행의 운영 미니칩(예약 N · 바인 N · 매출 N만)으로 상황을 한눈에.
  • 손님은 본인인증 회원만 예약 가능하며, 같은 닉네임 중복 예약은 자동 차단됩니다.

08QR · 체크인 — 방문 데이터의 시작

  1. 매장이용권/QR에서 고정 QR 3종(체크인·바인요청·이용권)을 선택 인쇄해 매장에 비치.
  2. 손님이 체크인 QR을 스캔하면 방문 기록 + 출석 도장(활동점수 +3, 7일 연속 시 +10 보너스). 4시간 내 중복 방지.
  3. 바인요청 QR을 스캔한 손님의 참가(바인) 요청은 장부에 알림으로 도착 — 승인만 하면 자동 기입됩니다.
✅ 체크인은 통계의 방문 집계·단골 분석(고객 CRM)에 자동 적재됩니다. 오픈 첫날부터 꼭 비치하세요.

09장부 — 운영의 심장

장부 시작

  1. 장부 › + 장부 추가 — 오늘 포스터가 있으면 자동 연동되어 게임명·바인이 채워집니다(직접 입력도 가능).
  2. 담당 직원(최대 10명)을 고르고 장부 시작 — 선택한 직원만 이 장부를 열람합니다.
  3. 시작 직후 "클락도 켤까요?"에서 바로 클락으로 이어갈 수 있어요. 메인 1개 + + 사이드로 같은 날 여러 게임 운영.

바인 기록

  • + 유저 추가(닉네임 입력 시 가입 회원 자동완성·✓회원 표시) → 행의 ＋를 눌러 바인 종류 선택:
  • 티켓/현금/이체/카드 × 완납/미수 · 가게지원 · 분납/할인 상세 입력. 얼리(더블/싱글)는 클락 레벨 기준 자동 판정.
  • 상단 지표(엔트리·완납 매출·생존·미수)와 하단 합계(총 엔트리·회수 티켓·완납 매출·미수금)가 실시간 계산됩니다.
  • 정렬(등록순/가나다/바인순)·검색·엑셀 내보내기 지원. 100명도 한 화면에서 관리됩니다.

마감

  • 레지 마감 — 신규 등록만 중단(정산과 별개).
  • 정산 마감 — 금일 통계 요약 확인 후 "꾹 눌러 마감 확정"(길게 누르기). 마감하면 읽기전용으로 잠기고(업주만 재오픈 가능), 클락이 연동돼 있었다면 최종 수치가 스냅샷으로 함께 저장됩니다.
  • 마감 후 🏆 순위 입력하기 버튼으로 바로 이어집니다.
⚠ 미수는 완납 처리 전까지 통계의 '미수금'에 계속 표시됩니다 — 마감 전 미수자 목록을 꼭 확인하세요.

10클락 — 토너먼트 타이머

  • 클락 진입 → 🎰 단독으로 시작(장부 없이) 또는 📒 장부 연동해 시작(오늘 장부 선택 — 게임명·엔트리·얼리 자동 반영).
  • 화면: LEVEL·타이머·BLINDS·ANTE·NEXT LEVEL·AVG STACK·PRIZE 보드·TOTAL PRIZE·RE-BUY/EARLY·REG CLOSE.
  • 블라인드는 프리셋(최대 50개) 저장/불러오기 + 게임 프리셋에서 가져오기.
  • ⤢ 전체화면은 매장 모니터용, 📺 TV 송출은 새 창(두 번째 화면)용. 공유 링크(?display=)로 아무 TV 브라우저에서나 열 수 있고 멀티게임 자동 순환·참가 QR·탈락 티커가 포함됩니다.
  • 장부 화면 상단의 클락 리모컨으로 레벨 이동·일시정지·아웃 처리·얼리 보정을 장부에서 바로 조작합니다.
  • 클락이 켜져 있으면 라이브 탭에 매장 게임이 실시간 공개되어 홍보 효과가 있습니다(30초 자동 갱신).

11순위 입력 — 입력 한 번으로 랭킹까지

  1. 순위 입력 → 날짜 선택 → 🎯 입력 중인 게임에서 메인/장부 게임/기타를 선택(하루 여러 게임이면 게임마다 따로).
  2. 📒 그날 장부 명단을 펼쳐 전체 추가 — 바인한 손님이 순위표에 자동으로 올라갑니다.
  3. 등수를 정리하고 프라이즈(선택)를 채운 뒤 순위 저장.
  • 등수마다 기준 점수(+N점)가 자동 부여되고, 닉네임이 일치하는 회원에게 활동점수가 자동 반영됩니다.
  • 손님 화면엔 닉네임 일부가 가려져 표시됩니다(예: 나*리). 일정탐색 '지난 대회'에 👑 우승자가 노출됩니다.
  • 인증 매장은 🖨 지류 양식(NAME/EVENT/PLACE/PRIZE/TD SIGN 수기 폼) 인쇄가 가능합니다.

12매장 랭킹 · 시즌

  • 매장 랭킹 — 순위 입력이 쌓이면 기간별 매장 랭킹과 🏆 역대 챔피언(명예의 전당)이 자동 구성됩니다.
  • 시즌 만들기 → 기간 내 순위점수(10/7/5/3/2/1)·상금이 자동 집계, 종료 시 상위 3명에게 활동점수(1000/500/300) 자동 보상 + 결과 아카이브.
  • 시즌은 만료일에 자동 종료되며, 매장 페이지의 랭킹 탭으로 손님에게도 공개됩니다.

13연합 리그 — 여러 매장 합동전

  1. 리그장이 리그를 만들고 닉네임으로 매장 초대 → 상대 매장이 수락하면 매칭.
  2. 이벤트 날, 각 매장은 자기 장부·순위로 예선을 치르고 상태 보드(🟡대기 🟢진행 🔴정산완료)로 실시간 공유.
  3. 매장별 정산 완료 보고(그날 순위→ITM 스냅샷) → 리그장이 전체 정산 완료 → 통합 ITM 공개.
  4. 파이널은 엔트리 최다 매장에 모여 새 클락(3테이블)으로 시작합니다.

14통계 — 정산과 분석

  • 통계 — 당일/일주일/한 달/총괄/요일별 탭 + 기간 직접 설정.
  • 총 엔트리·완납 매출·미수·회수 티켓·객단가(인/엔트리)·결제수단별 바인·할인 비율 + 클락 최종 보정 수치(마감 스냅샷).
  • 손님 유형(신규/기존/관계자) 분석 · 단골 TOP · ✨ AI 분석 요약 · CSV 내보내기.
  • 게임별 현황판(포스터·예약)에서 포스터별 바인 수·매출·마감·순위입력 여부를 확인.
  • 매주 월요일 아침, 지난주 주간 리포트가 자동 발송됩니다.

15매장이용권 — 디지털 티켓

  • 매장이용권/QR — 발급 한도(승인 매장 기본 1,000장) 내에서 손님에게 이용권을 발급합니다. 부족하면 충전 요청 → 운영자 승인.
  • 사용 경로: 손님 지갑의 QR 스캔 · 전화번호 조회 · 직접 차감. 바인 1회당 자동 적립 옵션도 있습니다.
  • 이용권으로 바인하면 장부에 '🎟 이용권 사용' 승인 요청이 떠서 업주가 확인 후 승인해야 기록됩니다(대납·수량 불일치 방지).
  • 손님끼리 양도 가능하며 이용 내역엔 실명(닉네임)이 함께 표시됩니다.

16직원 · 출근 관리

  • 직원 관리 — 이메일로 초대 → 직원이 수락하면 소속됩니다. 직급/호칭 설정 가능.
  • 직원은 장부·클락 등 운영 기능을 함께 쓸 수 있고, 장부는 담당 직원으로 지정된 사람만 열람합니다.
  • 출근 관리 — 직원 출근일지와 근무 스케줄을 기록·조회합니다.
  • 공동 사장님은 설정의 사장님 관리에서 닉네임으로 추가(운영자 승인제) — 대표 이전도 가능합니다(§19).

17공지 · 소통 · 리뷰

  • 매장 공지 — 매장 페이지에 고정 공지를 올립니다.
  • 📢 팔로워에게 알림 보내기(대시보드) — 마케팅 메시지를 팔로워 전원에게 푸시(하루 3회 한도).
  • 리뷰 — 체크인 인증 손님만 별점·후기를 남길 수 있고, 업주는 답글을 달 수 있습니다. 별점은 일정탐색 카드에 ⭐로 노출됩니다.
  • 매장 소통 탭에서 손님과 채팅형 소통이 가능합니다.

18대시보드 읽는 법

  • 지금 할 일 카드 — 상황 인지형 안내: 오늘 포스터가 있으면 "장부 시작하기", 진행 중인데 클락이 꺼져 있으면 "클락 켜기", 마감했는데 순위가 비면 "순위 입력하기"를 먼저 제안합니다. 이 카드만 따라가도 운영 흐름이 완성됩니다.
  • 🔴 라이브 운영 현황 — 진행 중 클락 + 대기 바인요청을 한 카드에서 처리.
  • 오늘 장부/클락 상태 · 최근 7일 추세 · 전주 대비 · 다가오는 예약 · 단골 TOP · 부스트/알림 보내기 바로가기.
  • 상단 📖 운영 가이드 바에서 이 설명서·슬라이드·PDF를 언제든 다시 열 수 있어요.
