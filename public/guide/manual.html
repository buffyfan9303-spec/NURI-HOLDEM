01📝 회원가입 — 일반 가입과 업주 가입의 차이

가입 창 여는 법

화면의 로그인 버튼을 누르거나, 로그인하지 않은 상태에서 글쓰기·댓글·예약 같은 기능을 시도하면 로그인 창이 자동으로 열립니다. 창 상단에 로그인 · 일반 가입 · 업주 가입 세 개의 탭이 있으니, 홀덤펍 사장님이라면 반드시 업주 가입 탭을 선택하세요.

로그인 창 › 업주 가입 탭

일반 가입 vs 업주 가입 비교

구분일반 가입업주 가입
입력 항목이름 · 닉네임 · 이메일 · 비밀번호 · 비밀번호 확인대표자명 · 닉네임 · 이메일 · 비밀번호 + 매장명 · 지역 · 연락처 · 상세 주소 · 사업자등록번호
가입 직후바로 로그인 가능로그인은 가능하지만 포스터 업로드는 운영자 승인 후 활성화
승인 기간없음영업일 기준 1~2일

닉네임 실시간 중복검사

닉네임을 입력하면 잠시 후 자동으로 검사 결과가 표시됩니다. 「✓ 사용 가능한 닉네임입니다」가 떠야만 가입 버튼이 활성화됩니다.

  • 규칙: 2~16자, 한글·영문·숫자·밑줄(_)·하이픈(-)만 가능
  • 「✗ 이미 사용 중」이면 다른 닉네임으로 바꿔야 합니다
  • 규칙에 맞지 않으면 노란색 안내 문구가 표시됩니다

동의 항목 (체크박스)

마케팅 정보 수신 (이벤트·할인·푸시)선택

맨 위의 「전체 동의」를 누르면 한 번에 모두 체크됩니다(선택 항목 포함). 각 약관 옆 「보기」를 누르면 전문을 읽을 수 있는 창이 열립니다. 필수 4개가 모두 체크되지 않으면 가입 버튼이 눌리지 않습니다.

⚠ 업주 가입 탭에는 비밀번호 확인란이 없습니다. 비밀번호(8자 이상)를 오타 없이 신중하게 입력하세요. 잘못 입력했다면 로그인 창의 「비밀번호를 잊으셨나요?」로 재설정하면 됩니다.
✅ 가입 완료 후 첫 로그인을 하면 휴대폰 본인인증을 진행하라는 안내가 나옵니다(1인 1계정 원칙). 업주는 「본인인증 + 운영자 승인」 두 가지가 끝나야 포스터 업로드가 열립니다.
🧨 만 19세 미만은 가입할 수 없습니다. 성인 확인에 체크하지 않으면 가입 자체가 차단됩니다.

02🔑 로그인 · 비밀번호 찾기 · 휴대폰 본인인증

로그인

  1. 로그인 창에서 가입한 이메일과 비밀번호를 입력합니다. 두 칸을 모두 채워야 로그인 버튼이 활성화됩니다.
  2. 성공하면 「환영합니다!」와 함께 창이 닫힙니다.

로그인이 안 될 때 나오는 안내는 두 가지입니다.

  • 「이메일 인증이 필요합니다」 — 가입 시 받은 인증 메일의 링크를 아직 누르지 않은 경우입니다. 받은 편지함(스팸함 포함)에서 인증 메일을 확인하세요.
  • 「이메일 또는 비밀번호를 확인해 주세요」 — 이메일이나 비밀번호가 틀린 경우입니다.

비밀번호 찾기 (로그인 창 → 「비밀번호를 잊으셨나요?」)

  1. 가입한 이메일을 입력하고 「인증번호 받기」를 누릅니다.
  2. 이메일로 온 인증번호(숫자, 최소 6자리)를 입력합니다. 숫자만 입력되며 최대 8자리까지 들어갑니다.
  3. 같은 화면에서 새 비밀번호(8자 이상)와 새 비밀번호 확인을 입력하고 「비밀번호 재설정」을 누릅니다.
  4. 완료되면 로그인 화면으로 돌아가 새 비밀번호로 로그인합니다.

메일이 안 왔다면 화면 하단의 「코드 재전송」으로 다시 받을 수 있습니다. 새 비밀번호 두 칸이 서로 다르거나 8자 미만이면 재설정이 진행되지 않습니다.

휴대폰 본인인증 — 왜 필요한가요?

1인 1계정 원칙과 안전한 거래를 위해 휴대폰 실명인증을 받아야 합니다. 본인인증을 하지 않으면 아래 기능이 잠깁니다. 시도하면 어떤 기능이 막혔는지 알려주는 안내 창이 뜹니다.

잠기는 기능설명
커뮤니티 글쓰기게시글 작성 불가 (대회 후기 포함)
중고장터 등록매물 등록 불가
대회 예약대회 좌석 예약 불가

본인인증 하는 법

프로필 관리 › 기본 정보 탭

  1. 프로필 관리 창을 열면 「휴대폰 본인인증」 안내 카드가 보입니다. 「휴대폰 본인인증」 버튼을 누르세요.
  2. 통신사 실명인증 창이 열립니다. 안내에 따라 인증을 완료합니다.
  3. 성공하면 「○○님 본인인증이 완료되었습니다」가 뜨고, 카드가 「✅ 본인인증 완료」(실명 표시)로 바뀝니다.
⚠ 한 사람은 하나의 계정에서만 인증할 수 있습니다. 이미 다른 계정에서 본인인증을 했다면 중복으로 판정되어 인증이 거절됩니다. 인증 창을 중간에 닫으면 「본인인증이 취소되었습니다」가 뜨며, 버튼을 다시 누르면 재시도할 수 있습니다.
✅ 업주는 승인 심사와 별개로 대표자 본인인증도 필요합니다. 가입 → 첫 로그인 시 본인인증 → 운영자 승인 순서로 마치면 포스터 업로드까지 전부 열립니다.

03👤 프로필 관리 — 기본 정보 · 보안 · 차단 · 탈퇴

기본 정보 탭

프로필 관리 › 기본 정보 탭

  • 프로필 사진: 동그란 사진 영역을 클릭해 이미지를 고릅니다. JPG·PNG·WEBP만 가능하고 최대 5MB입니다. 선택하면 정사각형으로 자르는 편집 창이 열립니다. 사진 우측 상단 ✕로 제거할 수 있고, 사진이 없을 때는 10가지 배경색 팔레트에서 색을 골라 이름 첫 글자 아바타로 표시됩니다.
  • 닉네임: 2~20자. 변경 후 30일 동안 다시 바꿀 수 없으며, 잠금 중에는 입력란이 비활성화되고 다음 변경 가능 날짜가 표시됩니다.
  • 받는 아이디 (매장이용권 수령용): 2~20자, 중복 불가. 최초 1회만 직접 설정할 수 있고, 설정 후에는 🔒 잠깁니다. 변경하려면 운영자에게 문의해야 하니 신중히 정하세요.
  • 이메일: 읽기 전용입니다. 변경할 수 없습니다.
  • 계정 요약: 내 등급(일반 회원 / 매장 업주 / 가게 직원 / 운영자)과 가입일이 표시됩니다. 업주는 「승인됨」 또는 「승인 대기」 상태를 여기서 확인할 수 있습니다.
  • 수정 후 반드시 「저장하기」를 눌러야 반영됩니다. 하단에는 고객센터 1:1 문의 버튼과 약관·정책 링크가 있습니다.

보안 탭 — 비밀번호 변경 (이메일 인증)

프로필 관리 › 보안 탭

  1. 새 비밀번호(8자 이상)와 확인을 입력합니다. 입력하는 동안 강도 게이지(매우 약함~강함)와 4가지 조건(8자 이상·영문·숫자·특수문자)이 실시간 표시됩니다.
  2. 「이메일로 인증코드 받기」를 누르면 가입 이메일로 인증번호가 발송됩니다.
  3. 메일로 받은 인증번호(숫자 6자리 이상)를 입력하고 「비밀번호 변경」을 누르면 완료됩니다. 「코드 재전송」도 가능합니다.
✅ 휴대폰에서 메일 앱을 확인하고 돌아오다 화면이 새로 열려도, 5분 안에 프로필 관리를 다시 열면 인증번호 입력 단계로 복귀합니다. 이때 새 비밀번호는 다시 입력해야 합니다(노란 안내 문구가 뜹니다).

푸시 알림 · 차단 목록

  • 푸시 알림 토글: 켜면 댓글·승인 소식·팔로우한 매장의 새 포스터를 브라우저 알림으로 받습니다. 지원하지 않는 브라우저에서는 토글이 비활성화됩니다.
  • 차단한 사용자: 게시글·중고장터 매물에서 「차단」을 누른 사용자 목록입니다. 차단하면 그 사용자의 글이 보이지 않으며, 여기서 「차단 해제」로 되돌릴 수 있습니다.

회원 탈퇴

  1. 보안 탭 맨 아래 「회원 탈퇴하기」를 누릅니다.
  2. 탈퇴 시 사라지는 항목 안내가 표시됩니다. 보유 중인 매장이용권 개수와 작성 글 개수도 함께 보여주니 꼭 확인하세요.
  3. 본인 확인을 위해 현재 비밀번호를 입력하고 「탈퇴하기」를 누르면 즉시 처리되고 자동 로그아웃됩니다.
🧨 탈퇴하면 실명·전화번호·본인인증 정보·생년월일 등 개인정보가 즉시 파기되고 계정이 폐쇄되어 복구할 수 없습니다. 보유 이용권도 함께 사라집니다. 매장 대표는 매장을 먼저 정리(삭제 또는 양도)해야 탈퇴할 수 있으며, 그 전에는 탈퇴가 거부됩니다.

04🏪 매장 만들기 — 신규 업주 첫 화면

업주 계정으로 로그인했는데 아직 등록된 매장이 없으면, 내 매장 탭에 들어가는 순간 자동으로 「내 매장 만들기」 화면이 나타납니다. 여기서 매장 정보를 한 번만 입력하면 NURI HOLDEM에 내 매장이 등록됩니다.

화면 상단의 📖 운영 가이드 먼저 보기 버튼을 누르면 새 창으로 사장님용 운영 가이드(포스터→장부→클락→정산 흐름)가 열립니다. 처음이라면 이걸 먼저 훑어보는 걸 권합니다.

05✅ 생성 직후 상태 · 운영자 승인 · 인증 매장 혜택

매장 생성과 관련해 서로 다른 두 단계가 있습니다. ① 매장 공개 승인(운영자가 매장 등록을 승인해야 손님에게 노출)과 ② 인증 매장 배지(승인과 별개로 운영자가 부여하는 신뢰 등급)입니다. 헷갈리기 쉬우니 구분해서 알아두세요.

생성 직후 — 승인 대기 중에도 운영은 가능

  • 매장을 만들면 내 계정에 매장이 바로 연결되어 내 매장의 관리 메뉴 전체(대시보드·포스터·장부·클락·직원 관리 등)를 즉시 사용할 수 있습니다. 승인을 기다리며 미리 세팅해 두면 됩니다.
  • 다만 일반 회원에게 보이는 것(일정탐색·커뮤니티의 매장 목록)은 운영자 승인 후부터입니다. 공개 목록에는 승인 완료 + 영업 중 상태인 매장만 올라갑니다.

06👥 업주 계정 vs 직원 계정 — 열리는 메뉴 차이

내 매장 탭의 메뉴 구성은 계정 종류와 부여받은 권한에 따라 달라집니다. 업주는 전체 메뉴가 열리고, 직원은 업주가 준 권한만큼만 보입니다.

직원 화면에서 벌어지는 일

  • 자물쇠 탭을 누르면 "접근 권한이 없습니다" 안내 화면이 뜨며, 업주에게 장부·순위 권한(장부) 또는 이용권 내역 권한(매장이용권/QR)을 요청하라고 알려줍니다.
  • 아직 소속 매장이 없는 직원은 "소속된 매장이 없습니다. 직원 승인 후 이용할 수 있습니다" 안내만 보입니다. 업주가 내 매장 › 직원 관리에서 이메일로 초대하고 직원이 알림에서 수락해야 소속됩니다.
  • 쓸 수 있는 메뉴가 하나도 없으면 "이 매장에서 사용 가능한 메뉴가 없습니다. 업주에게 장부 권한을 요청하세요"가 표시됩니다.
✅ 업주가 권한을 켜준 직후, 직원이 앱 화면으로 다시 돌아오기만 하면 권한이 자동으로 다시 확인되어 새 메뉴가 바로 열립니다. 재로그인이나 새로고침이 필요 없어요.

메뉴 사용 편의 기능(공통)

  • 즐겨찾기 ★: 메뉴 옆 별을 누르면 그 메뉴가 목록 맨 위에 고정됩니다. 매장당 최대 5개까지이며, 지금 쓰는 기기(브라우저)에 저장됩니다. 잠긴 메뉴에는 별이 없습니다.
  • 모바일은 현재 메뉴 한 줄만 보이는 접이식 메뉴(탭하면 전체 펼침), PC는 왼쪽 세로 사이드바로 표시됩니다.
  • 뒤로가기를 누르면 다른 화면으로 튕기지 않고 먼저 대시보드로 돌아옵니다.
⚠ 직원에게 장부 권한을 주면 장부뿐 아니라 순위 입력·매장 랭킹·연합 리그·클락·출근 관리까지 한 묶음으로 열립니다. 매출 숫자를 보여주고 싶지 않은 직원이라면 권한 부여 전에 이 점을 고려하세요.

07🔴 대시보드 상단 — 가이드 바·라이브 운영 현황

대시보드는 무엇인가요

내 매장 › 대시보드는 오늘 장부·클락·예약·출근·주간 추세를 한 화면에 모은 요약판입니다. 모든 카드는 누르면 해당 운영 화면으로 바로 이동합니다. 직원 계정은 부여받은 권한의 카드만 보이며, 권한이 하나도 없으면 카드 대신 "아직 부여된 권한이 없습니다" 안내만 표시됩니다(업주에게 장부·순위 또는 이용권 내역 권한을 요청하라는 문구).

맨 위 가이드 바 (📖 운영 가이드)

  • 사용설명서 — 지금 보고 계신 이 설명서가 새 탭으로 열립니다.

라이브 운영 현황 (운영 중에만 뜨는 카드)

장부·순위 권한이 있고, 클락이 진행 중이거나 손님의 바인(참가) 요청이 대기 중일 때만 나타나는 실시간 카드입니다. 평상시에는 보이지 않는 게 정상입니다.

구역내용
왼쪽 (클락)현재 블라인드(SB/BB)·레벨 번호·앤티, 휴식 중이면 BREAK. 오른쪽에 남은 시간이 1초 단위로 흐르며 초록=진행 중, 노랑=일시정지. 아래에 남은 인원(장부 엔트리+클락 인원 보정−탈락으로 자동 계산). 클락이 꺼져 있으면 "클락 꺼짐 — 눌러서 켜기"가 뜨고, 누르면 클락 화면으로 이동.
게임 탭메인+사이드 게임이 2개 이상 동시 진행이면 카드 위에 게임 전환 탭이 생깁니다. 일시정지된 게임은 ⏸ 표시. 보고 있던 게임이 끝나면 진행 중인 첫 게임으로 자동 전환됩니다.
오른쪽 (바인 요청)손님이 앱에서 보낸 참가 요청이 최대 3건 표시 — 이름·경과 시간(방금/몇 분 전)·요청 게임(메인/사이드). 3건이 넘으면 "외 N건 · 장부에서 전체 관리 →"로 이동.

대기 요청은 장부로 넘어가지 않고 이 카드에서 바로 처리할 수 있습니다.

  1. ✓ 짧게 탭 — 결제 기록 없이 승인만(손님이 게임에 추가됨). 결제는 나중에 장부에서 따로 기록.
  2. ✓ 길게 누르기(약 0.5초) — 결제까지 한 번에. 작은 창이 열리고 그 게임의 바인 금액(원)이 자동 입력됩니다(리바인·할인 시 직접 수정 가능). 현금/카드/이체 중 하나를 탭하면 전액 그 수단으로 기록되며 승인 완료. 자주 쓰는 결제수단이 맨 앞 초록 버튼으로 자동 정렬됩니다(사용할수록 이 기기에서 학습).
  3. 분할 버튼 — 현금/카드/이체 금액을 나눠 입력. 합계가 바인 금액과 일치하면 초록으로 표시되어 확인이 쉽습니다.
  4. ✕ — 요청 거절.
⚠ 결제 입력 합계가 0원이면 "금액을 입력하세요"가 뜨고 승인되지 않습니다. 또 승인·거절 처리 중에는 버튼이 잠깐 비활성화되어 중복 처리를 막습니다.

미니 추세 — 오늘 vs 평소

  • 삭제: [삭제]를 누르면 버튼이 [취소]/[삭제](모바일은 [삭제 확인])로 바뀌는 2단계 확인을 거칩니다. 실수로 한 번 눌러도 바로 지워지지 않습니다.
🧨 삭제 확인을 누르면 즉시 삭제됩니다. 예약을 받아둔 게임이라면 삭제 전에 예약 명단을 [CSV 내보내기]로 먼저 백업해 두세요.

12📋 게임 관리 — 예약 명단·운영 미니칩·장부·순위 경고

내 매장 › 게임 관리에는 내 계정으로 올린 포스터만 표시됩니다. 아직 게임이 없으면 [+ 첫 게임 등록하기] 버튼이 뜹니다.

목록 화면 읽는 법

  • 날짜 칩: 상단의 [전체 N] / 날짜별 칩(예: 7/21(월) 2 — 그 날짜 게임 수) / 📅 달력 직접 선택으로 원하는 날짜만 추려 관리합니다. 그 날짜에 게임이 없으면 "다른 날짜를 선택하세요" 안내가 뜹니다.
  • 운영 미니칩: 게임 제목 아래에 예약 N(예약자 수 — 새 예약·취소가 실시간 자동 반영), 바인 N·매출 N만(연결된 장부 기준 · 만원 단위), 마감 · 순위 ✓(장부 마감 + 순위 입력 완료)이 표시됩니다. 게임 관리가 곧 운영 현황판입니다.

예약 명단 — 누가 보고, 무엇을 할 수 있나

[예약관리](모바일은 하단 [예약 N] 버튼)를 누르면 명단이 펼쳐집니다. 예약자 이름·접수 시각이 담긴 이 명단은 업주 전용 화면이라 본인 매장 게임에서만 열립니다.

  • 같은 이름의 게임이 여러 날짜에 있으면 "같은 게임 · 날짜별 예약" 칩으로 날짜를 오가며 각 날짜의 예약을 확인합니다.
  • 예약자 한 줄: 순번 · 이름 · ✓ 방문(오늘 매장에 체크인한 손님과 자동 대조) · 단골 N회(우리 매장 예약 5회 이상) · 예약 접수 일시.
  • [수정] = 예약자 이름 고치기, [삭제] = 예약 빼기(노쇼·중복 등).
  • [CSV 내보내기] = 번호·예약자·예약시각을 엑셀에서 한글이 깨지지 않게 저장 — 파일명은 "게임명_월일_예약명단".
  • 단골에게만 [고객정보] 버튼이 떠서 바이인·방문·머니인·예약 횟수, 누적금액, 객단가를 바로 확인할 수 있습니다.
✅ 방문 체크는 손님 계정을 1순위로, 계정이 없으면 이름 일치로 대조합니다. 예약자 이름을 실제 체크인 이름과 같게 정리해 두면 방문 표시가 정확해집니다.

장부 바로가기

  • 연결된 장부가 없으면 버튼이 "장부 +" — 누르면 이 게임 정보가 미리 채워진 새 장부 등록으로 바로 이동합니다.
  • 연결 장부가 있으면 초록색 "장부 ▼" — 누르면 날짜·제목·진행중/마감 상태가 표시된 장부 목록(최신순)이 펼쳐지고, 한 줄을 누르면 그 날짜 장부가 열립니다.
  • 펼침 맨 아래 [+ 이 포스터로 새 장부]로 멀티데이·사이드 게임 장부를 추가할 수 있습니다(다른 날짜는 장부 화면에서 날짜를 바꾸면 됩니다).

"순위 미입력" 경고

연결 장부는 마감됐는데 그 날 순위가 아직 입력되지 않았으면 노란 경고가 뜹니다 — PC는 [순위 미입력] 배지, 모바일은 카드 하단 전체 폭의 "⚠ 순위 미입력 — 지금 입력하기" 띠입니다.

  1. 경고를 누르면 그 날짜의 순위 입력 화면으로 바로 이동합니다.
  2. 순위를 입력하면 랭킹·아카이브에 바로 반영되고, 경고가 사라지며 미니칩이 "마감 · 순위 ✓"로 바뀝니다.
⚠ 순위를 입력하지 않으면 그 게임이 손님 전적·랭킹 집계에 반영되지 않습니다. 장부를 마감한 당일에 순위까지 입력하는 습관을 들이세요.

13📋 게임 프리셋 — 만들기·수정·삭제와 저장 항목 전체

게임 프리셋이 무엇인가요?

매일 여는 단골 게임(데일리 토너먼트 등)의 내용을 템플릿으로 저장해 두는 기능입니다. 포스터나 장부를 만들지 않아도 게임 내용과 듀레이션, 블라인드 구조까지 미리 만들어 둘 수 있습니다. 날짜·시간은 프리셋에 저장되지 않습니다(게임을 열 때마다 달라지는 값이라서요).

내 매장 › 게임 프리셋

  • 이 메뉴는 업주(또는 운영자)에게만 보입니다. 직원 계정에는 메뉴 자체가 나타나지 않습니다.
  • 프리셋은 매장 단위로 저장되며, 목록은 최근에 수정한 것이 맨 위로 정렬됩니다.

사용법

  1. 목록 오른쪽 위 + 새 프리셋을 누릅니다.
  2. 프리셋 이름(필수, 최대 40자)을 입력합니다. 예: “데일리 6만 스타팅”. 이름이 비어 있으면 “프리셋 이름을 입력하세요”라며 저장되지 않습니다.
  3. 아래 항목을 채우고 저장을 누르면 목록에 추가됩니다. 저장 처리 중에는 버튼이 잠겨 이중 저장이 방지됩니다.
  4. 기존 프리셋은 목록에서 수정을 눌러 고치고, 삭제를 누르면 확인창(“○○ 프리셋을 삭제할까요?”)이 뜬 뒤 지워집니다.

14⏱ 블라인드 구조 편집기 — 레벨 추가·일괄 듀레이션·자동 생성

화면 구성

프리셋 편집 화면의 ‘블라인드 구조 (클락에 그대로 적용)’ 항목이 편집기입니다. 한 줄이 한 레벨이며 왼쪽부터 레벨 번호 · SB · BB · ANTE · 시간(분) · ✕(줄 삭제) 순입니다. 브레이크 줄은 번호 대신 B로 표시되고, 이름(기본 BREAK)과 시간만 입력합니다. 위에는 “총 ○레벨”이 표시되는데 이 숫자는 브레이크를 뺀 실제 레벨 수입니다.

레벨·브레이크 추가

  • + 레벨: 맨 아래에 새 레벨이 추가됩니다. 새 레벨의 SB는 직전 레벨의 BB, BB는 그 2배, 앤티는 BB와 같게 자동 계산되고, 시간은 직전 레벨과 같은 분으로 들어갑니다(첫 레벨이면 SB 100 / BB 200 / 20분). 그대로 두어도 되고 칸을 눌러 수정해도 됩니다.
  • + 브레이크: 8분짜리 BREAK 줄이 추가됩니다. 이름과 시간은 자유롭게 바꿀 수 있습니다.

듀레이션 일괄 설정

  1. 전체 ○분 → 전체 적용: 모든 레벨의 시간을 한 번에 같은 분으로 맞춥니다. 브레이크 시간은 건드리지 않습니다.
  2. 레벨 ○부터 ○분 → 적용: 지정한 레벨 번호부터 끝까지만 시간을 바꿉니다. 예: “전체 25분 적용 후, 레벨 16부터 15분 적용” → 1~15레벨 25분, 16레벨부터 15분짜리 구조가 됩니다. 여기서 레벨 번호는 브레이크를 뺀 번호 기준입니다.
  • 0이나 빈 값으로는 적용되지 않습니다(실수로 시간이 0분이 되는 것을 막는 안전장치).

자동 생성

  1. ‘자동 생성 · 최대 ○레벨’ 칸에 원하는 레벨 수(최대 60)를 입력합니다. 기본값은 18입니다.
  2. ⚙ 자동 생성을 누르면 표준 블라인드 표에 따라 SB/BB가 단계적으로 오르는 구조가 만들어집니다. 앤티는 BB와 같게, 레벨당 20분으로 들어가며, 5레벨마다 8분 브레이크가 자동 삽입됩니다. 표 범위를 넘는 깊은 레벨은 직전의 약 1.4배로 올려 깔끔한 단위로 반올림합니다.
  3. 이미 만들어 둔 구조가 있으면 “현재 블라인드 구조를 자동 생성으로 덮어쓸까요?” 확인창이 뜹니다. 확인하면 기존 구조가 통째로 교체됩니다.
✅ 추천 순서: ① 자동 생성으로 뼈대 만들기 → ② 일괄 듀레이션으로 앞뒤 구간 시간 조정 → ③ 개별 칸에서 세부 수정. 손으로 레벨을 하나하나 추가하는 것보다 훨씬 빠릅니다.
⚠ 편집기에서 구조를 아무리 잘 만들어도 화면 아래 ‘저장’ 버튼을 눌러야 프리셋에 남습니다. ✕로 화면을 닫으면 편집 내용이 사라집니다.

15🔁 프리셋은 어디서 재사용되나요?

토너먼트 클락에 블라인드 구조 그대로 적용

가장 중요한 재사용처입니다. 프리셋에 저장한 블라인드 구조를 클락 설정으로 한 번에 불러올 수 있습니다.

내 매장 › 클락 › 설정 화면 › 블라인드 구조 펼치기

  1. 클락 설정에서 ‘블라인드 구조 · ○레벨’ 줄을 눌러 펼칩니다.
  2. 맨 위 ‘📋 게임 프리셋에서 블라인드 불러오기’ 선택 상자를 엽니다. 목록에는 블라인드 구조가 저장된 프리셋만 이름과 레벨 수(예: “데일리 6만 스타팅 (18레벨)”)로 표시됩니다.
  3. 프리셋을 고르면 블라인드 구조 전체와 함께, 프리셋에 입력해 둔 스타팅·리바인·애드온 스택도 클락 설정에 채워집니다(비워 둔 스택 항목은 클락의 기존 값을 유지). 적용되면 “○레벨을 적용했습니다” 안내가 뜹니다.
  • 블라인드 구조 없이 만든 프리셋을 고르면 “이 프리셋엔 블라인드 구조가 없습니다”라고 안내되고 아무것도 바뀌지 않습니다.
  • 클락 설정 화면 자체가 업주/권한 있는 직원만 들어갈 수 있습니다.
⚠ 클락 설정 위쪽의 금색 ‘📋 프리셋 · 클릭해 불러오기’ 박스는 별도 기능인 클락 프리셋(클락 설정 전체를 통째로 저장한 것)입니다. 여기서 만드는 게임 프리셋과는 다른 저장함이니 헷갈리지 마세요. 게임 프리셋의 블라인드는 반드시 ‘블라인드 구조’ 펼침 안의 불러오기 상자에서 적용합니다.

나머지 항목의 활용

게임 제목·바이인·상금(GTD/비율)·듀레이션 문구·메모 등은 현재 다른 화면으로 자동 입력되지는 않습니다. 대신 프리셋 목록의 한 줄 요약으로 늘 보이므로, 포스터를 만들거나 장부에 게임을 적을 때 “우리 매장 표준 게임 조건”을 확인하는 기준표로 쓰면 됩니다.

✅ 운영 루틴 제안: 매장 대표 게임(데일리·주말 대회 등)마다 프리셋을 하나씩 만들어 블라인드 구조까지 저장해 두세요. 게임 시작 직전 클락 설정에서 프리셋만 고르면 블라인드·스택 입력이 몇 초 만에 끝납니다. 구조를 바꿨다면 프리셋도 함께 수정해 두어야 다음번에 옛 구조를 불러오는 실수를 막을 수 있습니다.

16📒 장부 목록·시작 설정·사이드 게임

장부가 하는 일

장부는 하루의 게임(메인 1개 + 사이드 여러 개)마다 참가자 명단과 바인(참가비) 결제 내역을 표로 기록하고, 매출·미수·엔트리를 자동 집계하는 전자 장부입니다. 여러 기기에서 같은 장부를 열어도 바인·명단·마감 상태가 실시간으로 동기화됩니다.

내 매장 › 장부

목록 화면 — 찾기·기간 필터·삭제

  • 검색창: 날짜·게임명으로 즉시 필터. + 장부 추가를 누르면 오늘 날짜 장부가 열립니다.
  • 기간으로 보기: 시작일~종료일을 지정하면 그 범위만 표시(종료일은 오늘까지만 선택 가능). 최근 7일 / 이번 달 원터치 버튼과 전체 보기 해제 버튼이 있습니다.
  • 목록은 날짜별 묶음으로 표시되며 최신 날짜만 펼쳐지고 과거 날짜는 접힙니다(헤더 탭으로 접기/펼치기). 헤더에 게임 수·진행 중·마감 수가 요약됩니다. 목록은 최근 90개 게임까지 불러옵니다.
  • 각 행: 메인/사이드 뱃지 + 게임명 + 바인 단가 + 담당 직원 이름 + 상태 뱃지(진행중=초록 / 레지마감=빨강 / 마감).
  • 휴지통(삭제): 업주·운영자에게만 보입니다. 확인창 후 진행하면 바인·명단·세션 기록이 전부 지워집니다.
🧨 장부 삭제는 되돌릴 수 없습니다. 하루 매출 기록이 통째로 사라지니, 잘못 만든 빈 장부를 정리할 때만 쓰세요.
⚠ 접근 권한: 담당 직원을 지정한 장부는 그 담당 직원과 업주·운영자만 열 수 있습니다(다른 직원에겐 🔒 잠김 표시). 담당을 아무도 지정하지 않은 장부는 장부 권한이 있는 직원 모두 열 수 있습니다. 또한 업주 승인이 완료된 계정만 장부를 시작·운영할 수 있습니다.

  • 세션에 할인 이벤트를 만들어 뒀다면 모달 상단에서 없음/할인1~5를 먼저 고르세요. 선택 즉시 "이 바인 0.5엔트리 = (10만−5만)/10만" 식으로 계산 결과를 보여줍니다.
  • 이미 입력된 셀을 다시 탭해 다른 수단을 고르면 덮어쓰기(수정)됩니다 — 미수를 나중에 받았을 때 같은 셀을 완납으로 다시 찍는 방식으로 회수 처리합니다.
  • 결제 취소(내역 삭제)는 업주가 설정한 취소 비밀번호를 입력해야 합니다. 비밀번호를 아직 설정하지 않았으면 취소 자체가 불가능하니 미리 설정해 두세요.

얼리 판정 규칙

  • 얼리는 첫 바인(1바인)에만 적용됩니다. 2바인부터는 리바인이라 항상 '없음'.

비고·플레이어 수정

  • 플레이어 이름 칸이나 비고 칸을 탭하면 수정 모달: 이름 수정(오타 교정 — 그 게임의 바인 기록도 함께 바뀜, 같은 이름이 이미 있으면 차단), 유형 변경, 비고(글자수 무제한).
  • 플레이어 삭제: 바인 0건이면 즉시, 바인이 있으면 취소 비밀번호를 입력해야 바인까지 함께 삭제됩니다.

이 게임의 클락이 오늘 장부에 연동돼 있으면(마감 전) 장부 안에 리모컨 바가 나타납니다. 클락 화면을 열어두지 않아도 제어되고, 시간이 지나 있으면 경과분만큼 레벨이 자동 전진해 표시됩니다.

3행얼리 보정 ± 스테퍼 — 클락이 자동 집계한 얼리 수에 수기로 가감

미수 관리

  • 미수 바인은 셀이 빨간색으로 표시되고, 상단 띠·하단 정산 바에 미수 합계(만원), 표 맨 오른쪽에 플레이어별 미수 합계 열이 있습니다. 티켓 미수는 금액이 아닌 장수로 하단에 따로 표기됩니다.
  • 미수를 받았으면 그 셀을 다시 탭해 완납으로 다시 선택하면 됩니다(덮어쓰기).
  • 마감 모달에는 미수자 명단이 금액 큰 순으로 정리되어 나옵니다.

레지 마감 — 신규 등록만 중단

하단 정산 바의 레지 마감을 누르면 새 유저 추가가 막히고 목록에 빨간 '레지마감' 뱃지가 붙습니다. 이미 등록된 손님의 바인 입력은 계속 가능하며(리바인 접수용), 레지 열기로 언제든 해제됩니다. 정산 마감과는 별개입니다.

정산 마감 — 꾹 눌러 확정

  1. 하단 정산 바(총 엔트리·회수 티켓·완납 매출·미수금 상시 표시)에서 정산 마감을 누릅니다.
  2. 요약 모달에서 총 바인 · 총 엔트리 · 완납 매출(만원) · 당일 미수금(만원) · 회수 티켓 · 가게지원 6개 수치와 미수자 명단을 확인합니다.
  3. 마감 메모(선택, 최대 300자 — 예: "미수 3건 내일 정산")를 적고 '꾹 눌러 마감 확정' 버튼을 길게 누르면 확정됩니다(오터치 방지).
  • 마감하면 그 게임 장부는 읽기전용이 됩니다 — 셀·명단·수정 모두 잠김. 마감 시각과 메모가 상단 띠에 표시됩니다.
  • 클락 연동 상태로 마감하면 클락 최종 수치(엔트리·생존·아웃·리바인·얼리·애드온)가 스냅샷으로 함께 저장되고, 클락 인원과 장부 인원이 다르면 마감 알림에 차이 경보가 붙습니다(정산 누수 조기 발견).
  • 마감 직후 알림 한 줄로 오늘 바인 수·매출(만원)·미수 건수를 요약해 줍니다.
  • 마감 후 다음 단계 바: 🏆 순위 입력하기(오늘 참가자 명단이 자동으로 넘어감, 미입력이면 강조 표시) / 📊 주간 리포트 보기.

마감 해제(재오픈)

마감 띠 오른쪽의 마감 해제는 업주(POS 관리 권한)에게만 보이며, 서버에서도 권한을 한 번 더 검사합니다. 해제하면 다시 입력 가능한 상태로 돌아갑니다.

엑셀 내보내기

하단 정산 바의 엑셀 버튼으로 현재 보고 있는 게임의 장부(세션 정보·명단·바인 내역)를 파일로 내려받습니다. 마감 전후 언제든 가능합니다.

✅ 마감 요약의 '완납 매출'은 실제로 받은 돈만 집계합니다. 미수·티켓·가게지원은 매출에 섞이지 않고 각각 따로 표시되므로, 시재 점검은 완납 매출 기준으로 맞추면 됩니다.
⚠ 실수로 마감했다면 직원은 풀 수 없습니다 — 업주 계정으로 '마감 해제'해야 합니다. 반대로 마감 전 장부는 누구든(권한 직원) 수정할 수 있으니, 하루 정산이 끝나면 바로 마감해 잠그는 습관을 권합니다.

19⏱ 클락 시작 방식·기본 설정·블라인드 편집·프리셋

클락이 하는 일

클락은 토너먼트 진행용 블라인드 타이머입니다. 레벨·브레이크가 시간에 맞춰 자동으로 넘어가고, 프라이즈·엔트리·평균 스택까지 한 화면에 보여줍니다. 상태가 서버에 저장되므로 사장님 휴대폰과 매장 TV 등 여러 기기에서 같은 클락이 실시간으로 똑같이 보입니다.

내 매장 › 클락 › 설정

클락 설정은 업주·권한이 있는 직원만 열 수 있습니다(권한이 없으면 안내 문구만 표시). 라이브 진행 중에는 화면 오른쪽 위 「설정」 버튼으로 들어갑니다.

시작 방식 — 단독 vs 장부 연동

방식설명
🎰 단독 클락장부 없이 타이머만 돌립니다. 엔트리·리바인은 하단 ± 버튼으로 직접 셉니다.
📒 장부 연동목록에서 장부(게임)를 고르면 게임명이 자동으로 들어오고, 엔트리·리바인·얼리가 장부 바인 기록에서 자동 집계됩니다. 시작 시 클락의 얼리 기준(분 환산)이 장부에도 기록돼 장부의 얼리 표시와 기준이 일치합니다.
  1. 맨 위 「클락 시작 방식」에서 단독 카드를 누르거나, 장부 목록에서 게임을 선택합니다. 목록은 최근 60개까지 표시되고 날짜·게임명으로 검색할 수 있으며, 사이드 게임·마감된 장부도 표시됩니다.
  2. 아래에서 기본 정보·얼리·블라인드·프라이즈를 설정합니다.
  3. 맨 아래 「📒 장부(…) 연동해 시작」 또는 「🎰 단독 클락 시작」을 누릅니다. 장부를 골라 놓고도 「단독으로 시작」 버튼으로 연동 없이 시작할 수 있습니다.
✅ 장부 탭에서 클락으로 넘어오면 그 게임이 미리 선택되고 게임명·애드온 설정이 자동으로 채워집니다.
⚠ 시작 직후 클락은 멈춘 상태(첫 레벨 시간 그대로)로 준비됩니다. 라이브 화면에서 ▶ START를 눌러야 카운트다운이 시작됩니다. 또한 이미 진행 중인 클락이 있을 때 「이 설정으로 다시 시작」을 누르면 현재 레벨·시간·보정값이 모두 처음으로 돌아갑니다.

얼리 구간 (레벨 기준)

「더블얼리 ~레벨까지」·「1얼리 ~레벨까지」를 레벨 번호로 입력합니다. 예) 더블얼리 1, 1얼리 4 → 레벨1 안에 도착하면 더블얼리, 레벨2~4 도착은 1얼리. 입력하면 몇 분에 해당하는지 자동 환산돼 표시되고, 블라인드 길이를 바꾸면 분 환산도 자동으로 다시 계산됩니다. 장부 연동 시 이 기준으로 얼리가 자동 분류되며, 라이브에서 수기 보정도 가능합니다.

블라인드 구조 편집

  • ⚙ 블라인드 자동 생성 — 등록마감·최대 레벨 기준으로 표준 블라인드 표를 자동으로 만듭니다. 기본 레벨 20분, 5레벨마다 8분 브레이크 자동 삽입, 등록마감 이후 구간은 블라인드 상승 폭이 더 가파릅니다. 실행 전 확인창이 뜨며 기존 구조를 덮어씁니다.
  • 📋 게임 프리셋에서 불러오기 — 「게임 프리셋」에 저장해 둔 블라인드 구조(블라인드가 들어 있는 프리셋만 목록에 표시)를 그대로 적용합니다. 프리셋에 스타팅·리바인·애드온 스택이 있으면 함께 적용됩니다.
  • 레벨별 편집 — 「블라인드 구조」를 펼치면 레벨마다 스몰·빅·안테·분을 직접 고치고 ✕로 삭제할 수 있습니다. 브레이크 줄은 이름과 분만 입력합니다.
  • + 레벨 — 마지막 레벨의 빅이 새 레벨의 스몰이 되고 빅은 그 2배, 안테는 빅과 같게, 시간은 직전 레벨과 동일하게 자동 추가됩니다. + 브레이크는 8분짜리 휴식이 추가됩니다.
  • 듀레이션 일괄 설정 — 전체 레벨(브레이크 제외)을 한 번에 몇 분으로, 또는 「N레벨부터 M분」으로 구간 적용. 「레지 마감 레벨부터로 설정」 버튼으로 시작 레벨을 바로 맞출 수 있습니다.

프라이즈

순위 이름(예: 1위)과 금액을 줄 단위로 추가·삭제합니다. 보드에는 입력한 숫자가 그대로 표시되고 TOTAL PRIZE는 합계로 자동 계산됩니다.

클락 프리셋 저장·불러오기

  1. 설정을 다 만든 뒤 이름(최대 30자)을 넣고 「저장」 — 이름을 비우면 대회명(없으면 ‘무제목’)으로 저장됩니다.
  2. 다음부터는 프리셋 목록에서 이름을 클릭 한 번으로 전체 설정을 불러옵니다. 검색 가능, 최근 수정 순 정렬, ✕로 삭제(확인창).
⚠ 프리셋은 매장당 최대 50개까지 저장됩니다. 초과하면 저장이 거부되니 안 쓰는 프리셋을 삭제하세요. 프리셋을 불러오면 현재 화면의 설정이 통째로 바뀝니다.

스택 계산식

TOTAL STACK = 엔트리×스타팅 스택 + 리바인×리바인 스택 + 애드온×애드온 스택 + 더블얼리×더블얼리 보너스 칩 + 1얼리×얼리 보너스 칩. AVG STACK = TOTAL ÷ 생존 인원(엔트리 − 아웃). 생존자가 0명이면 0으로 표시됩니다.

장부 연동 중 숫자가 채워지는 원리

  • 엔트리 = 장부에 바인 기록이 있는 참가자 수(같은 사람은 1명).
  • 리바인 = 같은 사람의 두 번째 바인부터 자동 카운트.
  • 얼리 = 바인 시각을 얼리 구간과 비교해 더블얼리/1얼리 자동 판정.
  • 장부에서 바인을 추가·수정하면 클락 화면에 실시간으로 즉시 반영됩니다. 화면 아래 안내줄에 「장부(날짜) 연동 중」이 표시됩니다.
✅ 자동 집계가 실제와 다르면 하단 Entries·Rebuy·Early·Addon ± 버튼으로 보정하세요. 보정값은 자동값에 더하고 빼는 방식이라 마이너스 보정도 가능합니다. 아웃 처리(Player)는 장부에 없으므로 반드시 클락에서 직접 눌러야 AVG STACK이 맞습니다.

토너 종료 — 입상 순위 자동 입력

마지막 레벨이 끝나면 종료 팡파르가 울리고, 장부 연동 + 운영 권한일 때 「🏆 입상 순위 입력」 창이 자동으로 뜹니다.

  1. 프라이즈 개수만큼 순위 줄이 미리 만들어집니다(프라이즈가 없으면 3줄). 「+ 줄 추가」로 늘릴 수 있습니다.
  2. 이름 칸은 장부 참가자 명단에서 자동완성됩니다.
  3. 상금 칸은 만원 단위입니다. 프라이즈에 원 단위 큰 숫자(1만 이상)로 입력해 두었다면 만원으로 자동 환산돼 미리 채워집니다.
  4. 「순위 저장」을 누르면 매장 순위·시즌 리그·머니인킹에 자동 반영됩니다. 이름이 입력된 줄이 최소 1줄은 있어야 저장됩니다.
⚠ 이 창을 닫아버리면 다시 자동으로 뜨지 않으니, 닫기 전에 순위를 저장하는 것을 권장합니다.

전체화면·TV 송출

  • ⤢ 전체화면 — 클락이 16:9 비율로 고정되어 어떤 TV·모니터·프로젝터에서도 같은 화면 배치로 커집니다. ESC 키나 휴대폰 뒤로가기로 해제되며, 뒤로가기를 눌러도 앱을 나가지 않고 전체화면만 풀립니다.
  • 📺 TV 송출 — 새 창으로 관전 전용 화면을 엽니다. 그 창을 매장 TV·빔프로젝터 화면으로 옮겨 띄워 두면 됩니다.

사이드 클락 (게임 여러 개 동시 운영)

클락 화면 맨 위에 「⏱ 게임 클락 N」 카드가 뜹니다. 게임 슬롯은 클락이 있는 게임 + 그날 장부에 만든 게임을 합쳐 보여주며, 타일마다 메인/사이드 이름·진행 상태(레벨 번호·브레이크·정지)·남은 시간·블라인드가 1초 단위로 갱신됩니다.

  1. 타일 탭 — 그 게임의 클락으로 전환합니다(현재 보고 있는 게임엔 ● 표시).
  2. ▶ 바로 시작 — 장부에는 게임이 있는데 클락이 없는 빈 슬롯을 탭하면, 메인 클락 설정을 복사해 오늘 장부에 연동한 채 즉시 시작합니다.
  3. ＋ 사이드 클락 — 확인창이 뜹니다. 확인 = 오늘 장부에 사이드 게임도 자동 생성(메인 게임 정보 복사, 이름은 「게임명 사이드N」) 후 클락 설정으로 이동 / 취소 = 장부 없이 클락만 설정으로 이동.
✅ 사이드 클락을 END 하면 자동으로 메인 클락 화면으로 돌아옵니다. 메인·사이드는 게임마다 클락이 1개씩이며 서로 독립적으로 진행됩니다.

22📺 TV 관전 디스플레이 열기 (송출 버튼·공유 링크·전체화면)

TV 관전 디스플레이란?

매장 TV·빔프로젝터에 띄워두는 보기 전용 큰 화면입니다. 대형 타이머, 레벨/블라인드/앤티, 다음 브레이크·등록마감 카운트다운, 상금 보드, 라이브 통계가 한 화면에 나오고, 클락 조작 버튼은 일절 없습니다(운영은 운영자 클락 화면에서만). 레벨이 넘어가거나 통계가 바뀌면 실시간으로 즉시 반영되고, 타이머는 종료 시각 기준으로 계산되기 때문에 화면을 잠깐 껐다 켜도 시간이 어긋나지 않습니다.

여는 방법 3가지

  1. 라이브 카드에서: 라이브 › 진행 중 게임 카드 › 📺 큰 화면(관전) 버튼을 누르면 같은 화면 위에 전체 오버레이로 열립니다.
  2. 운영자 클락에서: 운영자 클락 › 📺 TV 송출 버튼을 누르면 새 창으로 관전 화면이 열립니다. 그 창을 TV·프로젝터 쪽 화면으로 옮겨 띄워두면 됩니다.
  3. 공유 링크(주소)로: TV 송출로 열린 창의 주소를 그대로 복사해 두면, TV에 연결된 브라우저나 태블릿에서 그 주소만 입력해도 바로 관전 화면이 뜹니다. 주소에는 매장과 게임 번호가 들어 있고, 새로고침해도 관전 화면이 그대로 다시 표시되므로 즐겨찾기에 저장해두기 좋습니다. 게임 번호를 안 붙이면 메인 게임 기준으로 열립니다.

24🔄 자동 순환 · 참가 QR · 탈락 티커 · 스폰서 배너

멀티게임 자동 순환

메인 + 사이드 게임처럼 게임이 2개 이상 동시에 돌아갈 때만 상단에 🔄 자동/수동 버튼이 나타납니다.

  • 자동(기본값): 15초마다 다음 게임 화면으로 자동 전환되어 TV 한 대로 모든 게임을 번갈아 보여줍니다.
  • 수동 전환: 상단의 메인 / 사이드1 … 버튼을 직접 누르면 그 게임으로 고정되면서 자동 순환이 자동으로 꺼집니다. 다시 순환시키려면 🔄 버튼을 눌러 "자동"으로 되돌리세요.
  • 처음 열 때 지정한 게임이 없으면 진행 중인 게임 → 첫 번째 게임 순서로 알아서 보여줍니다.

참가 QR (스캔해서 참가)

화면 오른쪽에 초록색 "📲 스캔해서 참가" QR이 항상 표시됩니다. 손님이 휴대폰 카메라로 찍으면 지금 화면에 나오는 그 게임의 바인(참가) 요청이 접수되고, 운영자가 승인해야 실제 참가로 처리됩니다. 자동 순환으로 게임이 바뀌면 QR도 그 게임 것으로 함께 바뀝니다.

⚠ 게임이 여러 개일 때 손님이 QR을 찍는 순간 화면에 떠 있는 게임으로 참가 요청이 들어갑니다. "메인에 참가하세요"라고 안내할 거라면 자동 순환을 잠깐 수동으로 바꿔 메인 화면에 고정해 두는 게 안전합니다.

탈락 티커

지금 보고 있는 게임에서 탈락자가 나오면 화면 상단 중앙에 💥 방금 N명 탈락 · 남은 M명 배너가 5초간 번쩍 표시됩니다. 게임별로 따로 집계하기 때문에 자동 순환으로 화면이 다른 게임으로 넘어갔다 돌아와도 엉뚱하게 배너가 뜨지 않습니다. 운영자가 탈락 인원을 입력하는 즉시 실시간으로 반영됩니다.

스폰서 배너

화면 맨 아래에 스폰서·광고 이미지를 상시 노출할 수 있습니다. 별도 설정이 아니라 운영자 설정에 등록해 둔 클락 광고 이미지를 그대로 재사용합니다 — 이미지가 등록돼 있으면 자동으로 나오고, 없으면 배너 영역 자체가 표시되지 않습니다. 배너 높이는 화면의 약 9% 이내로 제한되어 타이머를 가리지 않습니다.

✅ 자주 하는 실수: ①QR로 참가 요청이 들어왔는데 운영자 화면에서 승인을 안 눌러 손님이 기다리는 경우 — QR 참가는 반드시 운영자 승인이 한 단계 더 있습니다. ②사이드 게임 시작 후 TV가 계속 메인만 보여주는 경우 — 이전에 수동으로 고정해 둔 것이니 🔄 버튼을 "자동"으로 되돌리세요.

28🏆 시즌 리그 (매장 랭킹 시즌 만들기·집계·종료·보상)

시즌 리그는 우리 매장 단독으로 여는 분기(기간제) 랭킹 대회입니다. 시즌 기간을 정해 두면, 그 기간에 순위 입력 메뉴로 등록한 대회 순위가 자동으로 합산되어 시즌 랭킹이 만들어집니다. 시즌 점수를 따로 입력할 필요가 전혀 없습니다.

관리 화면: 내 매장 › 매장 랭킹 (장부·순위 권한이 있어야 메뉴가 보이고, 시즌 시작/종료 버튼은 업주·관리 권한에게만 나타납니다). 손님에게는 매장 페이지 › 순위 탭에서 같은 시즌 랭킹이 조회 전용으로 공개됩니다.

시즌 시작하기

  1. 진행 중인 시즌이 없으면 화면 오른쪽 위에 + 시즌 시작 버튼이 보입니다. (진행 중 시즌이 있으면 이 버튼은 숨겨집니다 — 동시에 두 시즌을 열 수 없습니다.)
  2. 시즌 이름을 입력합니다(예: "2026 여름 시즌", 최대 40자). 이름을 비워 두면 "시즌 이름을 입력하세요" 오류가 뜨고 생성되지 않습니다.
  3. 시작일과 종료일을 고릅니다. 기본값은 시작일 = 오늘, 종료일 = 90일 뒤(분기)입니다.
  4. 시즌 시작을 누르면 즉시 개막되고 "시즌을 시작했어요" 안내가 뜹니다.

진행 중 시즌 화면

  • 시즌 이름, 기간, D-며칠(종료일 밤 11시 59분 기준 남은 날짜)이 표시됩니다.
  • 랭킹은 상위 20명까지 표시되며, 각 줄에 순위(1~3위는 메달 색), 닉네임(실명 병기), 참가 횟수 · 최고 순위 · 시즌 점수가 나옵니다.
  • 순위 입력에서 새 순위를 등록하는 순간 새로고침 없이 실시간으로 시즌 랭킹이 갱신됩니다.
  • 기록이 아직 없으면 "순위가 등록되면 집계됩니다" 안내만 표시됩니다.

시즌 종료와 보상

  • 자동 종료: 종료일이 지나면 자동으로 마감되고 보상이 지급됩니다.
  • 수동 종료: 진행 중 시즌 카드의 시즌 종료 버튼(업주·관리 권한 전용)을 누르면 확인 창이 뜹니다 — "상위 3명에게 보상(활동점수)을 지급할까요? 되돌릴 수 없습니다."
  • 종료 시 상위 3명에게 활동점수 1000 / 500 / 300점이 자동 지급되고, 전체 순위 기록이 그대로 스냅샷으로 보관(아카이브)됩니다. 완료되면 "시즌 종료 · N명 기록 아카이브 + 상위 보상 지급"이라고 알려줍니다.
🧨 시즌 종료는 되돌릴 수 없습니다. 종료 즉시 보상이 지급되고 기록이 확정되므로, 그날 순위 입력을 모두 마친 뒤에 종료하세요.
✅ 시즌 점수의 원천은 오직 "순위 입력"입니다. 평소처럼 대회 순위만 꼬박꼬박 등록하면 시즌 랭킹·보상·아카이브가 전부 자동으로 굴러갑니다.

⚠ 아카이브는 시즌 종료 시점의 스냅샷입니다. 종료 후에 과거 날짜로 순위를 고쳐도 이미 끝난 시즌의 기록·보상은 바뀌지 않으므로, 정정할 것이 있다면 반드시 종료 전에 처리하세요.

30🤝 연합 리그 (매장 간 공동 리그: 생성·초대·정산·파이널)

연합 리그는 여러 홀덤펍이 하나의 공동 랭킹 보드를 함께 운영하는 기능입니다. 흐름은 리그 생성 → 매장 초대 → 상대 수락 → 각 매장 포인트 기록 → 통합 순위이고, 대회 당일에는 실시간 정산 보드로 매장별 진행 상황을 모으고 파이널(집결전)까지 진행합니다.

위치: 내 매장 › 연합 리그 — 장부·순위 권한이 있어야 메뉴가 보이며, 리그 만들기·초대·수락/거절·해산·정산 조작은 업주(관리 권한)만 가능합니다. 포인트 기록은 장부·순위 권한 직원도 할 수 있습니다.

생성과 초대

  1. "새 연합 리그 만들기"에 리그 이름(최대 20자)을 넣고 + 만들기. 이름이 비면 오류가 뜹니다. 만든 매장이 리그장이 됩니다.
  2. 리그 카드의 매장 선택 목록(이미 초대했거나 소속된 매장은 목록에서 자동 제외)에서 매장을 고르고 초대 — 상대 매장 전원에게 알림이 갑니다. 같은 매장을 다시 초대하면 "이미 초대한 매장입니다" 오류가 뜹니다.
  3. 초대받은 매장 화면 상단에는 "받은 초대" 박스가 뜨고, 수락/거절 버튼은 그 매장 업주만 누를 수 있습니다. 응답하면 리그장 매장에 알림이 갑니다.
  4. 리그장은 아직 수락 전(대기/거절) 매장의 배지에 있는 ×로 초대를 취소할 수 있습니다. 멤버 배지에는 수락(초록)·대기(노랑)·거절(빨강) 상태가 그대로 표시됩니다.
🧨 리그 카드 오른쪽 위 × (리그 해산) 버튼은 확인 창 없이 즉시 리그가 통째로 해산됩니다(리그장 업주 전용). 누르기 전에 한 번 더 생각하세요.

통합 순위와 포인트 기록

  • 아래 입력줄에 이름(최대 30자) + 포인트를 넣고 기록을 누르면 우리 매장 명의로 리그 장부에 적립됩니다. 이름이나 포인트가 비어 있거나 0이면 "이름과 포인트를 입력하세요" 오류가 뜹니다. 음수 입력으로 차감도 가능합니다. 실패 시 "멤버 수락 상태와 장부 권한을 확인하세요"가 뜹니다 — 수락 전 매장은 기록할 수 없습니다.
  • 통합 순위(TOP 10)는 리그 시즌 시작일 이후의 기록을 이름 기준으로 전 매장 합산합니다(앞뒤 공백·대소문자는 무시하고 같은 사람으로 취급). 합계가 0점 이하인 사람은 표에서 빠지며, 각 줄에 몇 개 매장에서 뛰었는지도 표시됩니다.
  • "전체 입력 내역 보기"를 펼치면 최근 기록(최대 400건)이 날짜·매장·이름·±포인트로 나오고, 우리 매장이 넣은 기록과 리그장은 모든 기록을 ×로 삭제할 수 있습니다(삭제도 확인 창 없이 즉시 처리).
⚠ 통합 순위는 이름 글자로 사람을 묶습니다. 매장마다 같은 손님을 "홍길동"/"길동이"처럼 다르게 적으면 점수가 갈라지니, 리그 공용 닉네임 표기를 매장끼리 미리 통일하세요.

📡 실시간 정산 보드 (대회 당일)

  • 참가 매장 = 리그장 + 수락한 매장. 매장별로 🟡 시작 전 / 🟢 진행 중 / 🔴 정산 완료 상태와 엔트리 수가 모든 참가 매장에 실시간으로 보입니다. 리그 전체 상태 배지(대기·진행 중·정산 완료·파이널)도 함께 표시됩니다.
  • 내 매장 보고(참가 매장 업주, 파이널 전 단계까지): 엔트리 수를 입력하고 🟢 진행 또는 🔴 정산완료(ITM 보고)를 누릅니다. 되돌리기로 시작 전 상태로 복구할 수 있습니다.
  • 정산완료를 누르는 순간, 오늘(한국 시간 기준) 우리 매장 "순위 입력"에 등록된 순위 중 상금이 적힌 사람만 골라 이름·등수·상금이 입상(ITM) 명단으로 자동 보고됩니다. 완료 시 "정산 완료 — 입상 N명 보고됨"이 뜹니다.
  • 리그장 전용 · 전체 정산 완료: 모든 참가 매장이 🔴 정산 완료가 되어야 버튼이 활성화됩니다(그 전에는 "전체 정산 대기 (몇/몇 정산)"으로 잠김). 누르면 엔트리가 가장 많은 매장이 파이널 집결 매장으로 자동 확정됩니다.
  • 전체 정산 후에는 통합 ITM 명단(전 매장 입상자를 등수순 정렬, 매장·상금 표시)과 총 엔트리, 파이널 집결 매장 안내("3테이블로 시작, 통합 클락은 집결 매장에서 새 클락으로 진행")가 모두에게 공개됩니다.
  • 리그장은 🏁 파이널 시작으로 파이널 단계에 진입시키고, 초기화 버튼(확인 창 있음)으로 이번 이벤트 현황 전체를 리셋해 다음 이벤트를 준비할 수 있습니다.
✅ 정산완료를 누르기 전에 반드시 내 매장 › 순위 입력에서 오늘 대회 순위(상금 포함)를 먼저 등록하세요. 순위에 상금 칸이 비어 있으면 그 사람은 입상 명단에 올라가지 않아 "보고된 입상 명단이 없습니다"로 나옵니다.

34🎟 매장이용권 발급 — 수량·받는 손님 지정·한도 충전

매장이용권이란

손님에게 나눠주는 매장 전용 쿠폰입니다. 종이 쿠폰 대신 손님 앱 지갑에 들어가고, 손님이 매장에서 사용하면 사장님 화면에 실시간으로 찍힙니다. 매장이용권은 금전적 가치(현금 가치)가 없으며, 화면에도 이 문구가 항상 표시됩니다.

내 매장 › 매장이용권 관리 — 대시보드의 이용권 카드에서 열거나, 매장관리 메뉴 안의 같은 패널에서 관리합니다.

발급 방법

  1. ‘매장이용권 발급’ 접힌 칸을 눌러 펼칩니다. 제목 옆에 잔여 한도 ○개가 함께 보입니다(50개 미만이면 빨간색 경고 표시).
  2. 이용권 이름을 입력합니다(기본값 ‘매장이용권’, 예: 데일리 1회 참가권).
  3. 수량을 정합니다: 1~1,000개. −/＋ 버튼은 꾹 누르고 있으면 점점 빨라져 연타할 필요가 없고, 숫자를 직접 입력해도 됩니다.
  4. (선택) 받는 손님 지정: ‘아이디(닉네임)로 지정’ 또는 ‘전화번호로 지정’ 버튼을 누르고 입력하면 잠시 후 후보가 자동완성으로 뜹니다(키보드 ↑/↓로 고르고 Enter로 선택 가능). 입력창 위에는 최근 발급했던 단골 손님 최대 6명이 원탭 버튼으로 떠서 바로 고를 수 있습니다.
  5. ‘＋ ○개 발급’ 버튼을 누르면 즉시 배포됩니다. 손님을 지정했다면 버튼에 ‘→ 받는 손님 이름’이 함께 표시됩니다.
받는 손님결과
회원 지정그 회원의 앱 지갑으로 바로 전달(양도) — 손님이 앱에서 직접 사용
미지정‘매장 보관’ 상태로 발급 — 매장에서 보관·관리

제한·조건

  • 발급 권한: 업주(본인 매장) 전용입니다. 인증 직원은 열람·사용 처리만 가능하며, 발급 칸 대신 안내 문구가 보입니다.
  • 운영자 승인제: 운영자가 이 매장의 발급을 승인하기 전에는 노란 경고(“운영자 승인 후 발급 가능”)가 뜨고 발급 버튼이 눌리지 않습니다.
  • 본인인증 회원만: 미인증 회원은 후보 목록에 ‘미인증 · 발급 불가’ 빨간 배지가 붙고 선택 자체가 막힙니다. 눌러도 “본인인증을 완료한 회원에게만 발급할 수 있습니다” 안내가 뜹니다.
  • 1회 최대 1,000개, 그리고 매장의 잔여 발급 한도 안에서만 발급됩니다.

발급 한도 충전(구매) 요청

  1. 발급 칸 아래 ‘🛒 발급 한도 충전(구매) 요청’을 누릅니다. 한도가 부족한 상태에서 발급을 시도하면 이 창이 자동으로 열립니다.
  2. 수량을 고릅니다: 500 / 1,000 / 3,000개 버튼 또는 직접 입력(1~100,000개).
  3. (선택) 요청 메모를 적습니다(최대 80자, 예: 주말 이벤트용).
  4. ‘충전 ○개 요청’을 누르면 운영자에게 전달되고, 운영자가 승인해야 한도가 실제로 충전됩니다. 구매·정산은 운영진이 별도 안내합니다.
  • 대기 중 요청이 있으면 “⏳ 충전 요청 ○개 — 운영자 승인 대기 중”으로 표시되고, 처리 전에는 새 요청을 추가로 넣을 수 없습니다(서버가 거부).
  • 최근 처리 결과 2건(✅ 승인 / ❌ 거절 + 운영자 메모)이 아래에 남습니다.
✅ 자주 주는 단골에게는 ‘최근’ 원탭 버튼이 가장 빠릅니다. 단골 TOP 화면에서 ‘이용권 보내기’로 들어오면 받는 사람이 자동 입력되고, 1명만 일치하면 즉시 선택까지 끝나 있습니다.
⚠ 받는 손님을 지정하지 않고 발급하면 손님 지갑이 아닌 ‘매장 보관’으로 쌓입니다. 손님에게 직접 주려면 반드시 아이디(닉네임) 또는 전화번호로 지정하세요.

35📒 이용권 사용(3가지 방법)·이용 내역·보유자 현황

손님이 이용권을 쓰는 3가지 방법

사장님이 방법을 알아야 손님에게 안내할 수 있습니다. 어느 방법이든 사용 즉시 사장님의 이용 내역에 실시간으로 표시됩니다.

방법손님이 하는 일비고
① 매장 QR 스캔앱에서 ‘사용하기’ → 매장에 비치된 ‘매장이용권 사용 QR’ 스캔발급한 매장에서만 사용 가능. 매장 QR이 아니면 “매장 QR이 아닙니다”로 거부
② 전화번호 입력발급 매장 업주의 전화번호를 입력해 사용QR을 못 찍는 상황의 대안
③ 직접 전송‘전송’ 버튼 한 번으로 발급 매장에 바로 사용“되돌릴 수 없습니다” 확인창 후 확정
✅ 손님이 이용권을 사용하면 매장 방문이 자동으로 기록되어 방문 횟수·단골 집계에도 반영됩니다.

이용 내역 — 실시간, 실명 표기

내 매장 › 매장이용권 관리 › 🎟 이용 내역 — 패널 맨 위에 기본으로 펼쳐져 있습니다.

  • ↗ 발급(보냄)과 ↘ 사용(받음)이 한 줄씩 최신순으로 흐르고, 발급·사용이 생기면 새로고침 없이 즉시 나타납니다(수동 ↻ 새로고침 버튼도 있음).
  • 이름은 실명/닉네임을 함께 표기합니다 — 닉네임만으로는 동명이인을 구분할 수 없기 때문입니다. 받는 사람이 없으면 ‘매장 보관’으로 표시됩니다.
  • 같은 시각(분 단위)·같은 손님·같은 이용권은 ×10처럼 한 줄로 묶여 10장 발급이 10줄로 도배되지 않습니다. 최근 30줄까지 보입니다.
  • 이 내역은 업주뿐 아니라 권한을 받은 인증 직원도 열람할 수 있습니다.

보유자 현황·통계 (업주 전용)

‘📊 보유자 현황·통계’ 접힌 칸을 눌러 펼칩니다.

  • 요약 카드 3개: 👥 보유 회원 수 / 🎟 활성 이용권(발급된 것과 사용된 것의 합계) / ✨ 잔여 이용권(아직 안 쓴 것). 아래에 사용률 % 게이지가 함께 표시됩니다.
  • 사용처 TOP: 우리 매장이 배포한 이용권이 실제 사용된 매장 순위(최대 6곳). 우리 매장이 아닌 곳에서 쓰였으면 ‘타 매장’ 표시가 붙습니다.
  • 보유자 목록: 보유 인원·보유 갯수 합계가 위에 뜨고, 실명·닉네임으로 검색할 수 있습니다. 각 줄에 보유 ○개·사용 ○회가 보이고, ‘관리’를 누르면 그 손님의 이 매장 이용내역(이용권 이름 + 날짜·시간, 최근순)이 펼쳐집니다.

삭제(✕)

  1. 보유자 줄 오른쪽 ✕를 누릅니다.
  2. “○○의 이용권 ○개를 완전히 삭제할까요? 되돌릴 수 없습니다.” 확인창에서 확인하면 그 묶음(보유분+사용분)이 모두 삭제됩니다.
⚠ 업주가 직접 삭제할 수 있는 것은 ‘매장 보관’ 묶음뿐입니다. 이미 회원 지갑으로 보낸 이용권의 삭제 버튼은 운영자에게만 보입니다.
🧨 삭제는 되돌릴 수 없습니다. 확인창의 인원·개수를 반드시 읽고 진행하세요.

인쇄해서 매장에 비치하기

  1. QR 칸 아래 ‘인쇄할 QR 선택’에서 체크 버튼으로 고릅니다. 종이가 작아 한꺼번에 다 넣을 수 없으니 1~3개 선택을 권장합니다.
  2. 선택지는 총 7종: 🎟 매장이용권 사용 / 📍 출석 체크인 / 📱 회원가입 / 🙋 바인(참가) 요청 / 🏆 바인 요청·메인 / 🎲 사이드1 / 🎲 사이드2.
  3. ‘🖨 선택한 QR 출력해 매장에 비치’를 누르면 새 창에 인쇄용 페이지(NURI HOLDEM 로고·문구 포함, 고화질 QR)가 열리고 인쇄 대화상자가 자동으로 뜹니다.
  • 테이블별 바인 QR(메인·사이드1·사이드2)은 인쇄물에 ‘🪑 ○○ 테이블’ 띠가 붙고, 스캔하면 그 게임으로 바로 참가 요청이 들어옵니다 — 테이블마다 한 장씩 비치하는 용도입니다.
  • 하나도 선택하지 않고 누르면 “인쇄할 QR을 1개 이상 선택하세요”가 뜹니다.
⚠ 브라우저 팝업 차단이 켜져 있으면 인쇄 창이 열리지 않습니다. “팝업이 차단되었습니다” 안내가 뜨면 팝업을 허용한 뒤 다시 시도하세요.
✅ 이용권 사용 QR과 출석 체크인 QR은 고정 QR이라 한 번 인쇄해 코팅해두면 계속 씁니다. 다시 뽑을 필요가 없습니다.

방문 체크인 모달 — 예약·방문 체크

내 매장 › 예약·방문 체크 — 업주·직원이 체크인 QR과 오늘 방문자를 한 화면에서 봅니다.

  1. 모달을 열면 큰 고정 체크인 QR이 표시됩니다. 손님이 스캔하면 우리 매장에 방문 체크됩니다.
  2. ‘체크인 링크 복사’ 버튼으로 같은 링크를 복사해 카톡 등으로 보낼 수도 있습니다.
  3. 아래 ‘오늘 방문 ○명’ 목록에 체크인한 손님이 실시간으로 쌓입니다(오늘 0시부터 집계).
  • 제한: 로그인한 회원만 체크인할 수 있고, 4시간 안에는 중복 체크인이 막힙니다. 손님이 매장이용권을 사용하면 별도 스캔 없이도 방문이 자동 기록됩니다.
  • 목록의 각 손님에게는 방문 횟수 배지가 붙습니다: 첫 방문(초록) / ○회 / 단골 ○회(5회 이상, 금색) — 시각(시:분)도 함께 표시됩니다.
✅ 입장 동선(카운터)에 출석 체크인 QR을, 각 테이블에 해당 테이블의 바인 요청 QR을 비치하면 출석 적립과 참가 접수가 모두 자동으로 굴러갑니다.

37👥 직원 초대·직책·권한·제거

구성원 초대 (이메일)

매장에서 함께 일할 직원을 이메일로 초대해 구성원으로 등록합니다. 이 메뉴는 업주(및 플랫폼 운영자)에게만 보이며, 직원에게는 아예 표시되지 않습니다.

내 매장 › 직원 관리 › 구성원 목록 — 직원 관리 화면은 아코디언(접이식) 구조로, 처음 들어가면 「구성원 목록」이 펼쳐져 있습니다. 한 번에 하나의 항목만 펼쳐집니다.

  1. 「구성원 초대」 칸에 초대할 회원의 가입 이메일을 입력하고 [초대] 버튼을 누릅니다. (이메일이 비어 있으면 버튼이 눌리지 않습니다)
  2. 초대가 발송되면 「대기중 초대」 목록에 이름·이메일과 함께 수락 대기로 표시됩니다. 옆의 [취소]로 언제든 초대를 거둘 수 있습니다.
  3. 상대방이 본인 알림에서 수락하면 그때 구성원이 됩니다. 거절할 수도 있습니다.
⚠ 초대 대상은 먼저 NURI HOLDEM에 일반 회원으로 가입돼 있어야 합니다. 가입하지 않은 이메일로는 초대가 실패합니다. 또한 수락 전(대기중) 상태에서는 아직 구성원이 아니므로 직책·권한을 줄 수 없습니다.

직책(호칭) 지정

구성원 목록의 각 직원 카드에서 직책 칸에 입력하면 됩니다. 입력 칸을 벗어나거나 엔터를 치면 자동 저장됩니다(최대 20자). 매니저·플로어·딜러·칩러너·매장장·직원 등 추천 목록에서 골라도 되고 직접 써도 됩니다.

✅ 직책은 표시용 라벨일 뿐이며 아무 권한도 부여하지 않습니다. "매니저"라고 적어도 장부는 열리지 않습니다 — 권한은 아래 토글로 별도로 켜야 합니다.

권한 부여 (토글 2종)

각 직원 카드의 권한 버튼을 눌러 켜고 끕니다. 켜짐/꺼짐이 즉시 화면에 반영되고, 실패하면 원래대로 되돌아가며 오류 안내가 뜹니다.

장부·순위 권한장부 · 순위 입력 · 매장 랭킹 · 연합 리그 · 클락 · 출근 관리

권한이 없는 직원에게도 「장부」·「매장이용권/QR」 메뉴는 자물쇠 표시로 보이며, 눌러보면 "업주에게 권한을 요청하세요" 안내가 나옵니다. 포스터·통계·직원 관리·설정처럼 업주 전용 메뉴는 직원에게 아예 숨겨집니다.

✅ 권한을 켠 직후 직원이 새로고침을 안 해도, 직원이 앱 화면(창)을 다시 열어 포커스만 주면 권한이 자동으로 다시 확인되어 즉시 반영됩니다.

구성원 제거

  1. 직원 카드 오른쪽 위 [제거]를 누릅니다.
  2. "구성원을 제거하시겠습니까? (일반 회원으로 전환)" 확인창에서 확인하면 완료됩니다.
⚠ 제거된 직원은 계정이 삭제되는 게 아니라 일반 회원으로 전환됩니다. 매장 메뉴 접근만 사라지며, 다시 쓰려면 이메일로 재초대해야 합니다.

38🕒 출근 스케줄·출근일지·셀프 출퇴근

딜러 출근 스케줄 (업주가 짜는 월간 근무표)

월 캘린더에 직원을 날짜별로 배정하고 출근·퇴근 시각을 기록하는 근무표입니다. 근무시간이 자동 집계되어 인건비 정산의 기초가 됩니다.

내 매장 › 직원 관리 › 딜러 출근 스케줄

  1. 상단 ‹ › 버튼으로 월을 이동합니다. [이번달] 버튼으로 바로 돌아올 수 있습니다.
  2. 직원 등록 — 초대로 등록된 매장 구성원은 명부에 자동 포함됩니다. 구성원이 아닌 알바 등은 이름을 직접 입력해(최대 20자) [+ 추가]로 명부에 올릴 수 있습니다.
  3. 캘린더에서 날짜를 누르면 아래에 편집 칸이 열립니다. 직원 이름 버튼을 누르면 그날 출근으로 배정(다시 누르면 해제)되고, 배정된 직원은 출근·퇴근 시각을 입력할 수 있습니다.

직원 출근일지 (열람 전용)

내 매장 › 직원 관리 › 직원 출근일지 — 월별 출퇴근 기록을 최신 날짜순 목록으로 보여줍니다. 날짜·이름·출퇴근 시각·근무시간이 한 줄씩 표시되며, 직원이 출퇴근을 찍는 즉시 실시간으로 반영됩니다. 여기서는 수정할 수 없고, 수정은 스케줄 화면에서 합니다.

직원 셀프 출퇴근 (직원 본인 화면)

직원이 자기 출퇴근을 직접 찍는 화면입니다. 내 매장 › 출근 관리

  1. 업주가 스케줄에 배정한 본인 일정 목록이 뜹니다. 확정된 스케줄에는 "확정" 표시가 붙습니다.
  2. 오늘 날짜에만 [지금 출근]·[지금 퇴근] 원터치 버튼이 나타나 현재 시각이 바로 기록됩니다.
  3. 다른 날짜는 출근·퇴근 시각 칸을 직접 고쳐 기록할 수 있습니다.
⚠ 두 가지 조건이 맞아야 직원 화면에 일정이 보입니다. ① 직원에게 장부·순위 권한이 있어야 「출근 관리」 메뉴 자체가 열립니다. ② 스케줄에 적은 배정 이름이 그 직원의 실명 또는 닉네임과 정확히 일치해야 본인 일정으로 인식됩니다. 별명·오타로 배정하면 직원 화면에 아무것도 안 뜹니다.
✅ 직원이 찍은 출퇴근은 업주의 스케줄·출근일지 화면에 실시간으로 넘어옵니다. 근무시간은 출퇴근이 모두 기록된 날만 급여 계산에 잡히므로, 퇴근 누락이 없는지 월말에 출근일지를 한 번 훑어보세요.

39🤝 공동 사장님 추가·제외·대표 이전

공동 사장님(공동 업주)이란

한 매장을 여러 명의 사장이 함께 운영할 수 있게 하는 기능입니다. 승인된 공동 업주는 장부·포스터·이용권을 업주 권한으로 함께 관리합니다. 직원 권한과 달리 플랫폼 운영자(NURI HOLDEM 관리자)의 승인을 거쳐야 활성화되는 승인제입니다.

내 매장 › 설정 › 사장님(공동 업주) 관리 — 설정 메뉴는 업주(및 플랫폼 운영자)에게만 보이며, 이 카드도 업주만 볼 수 있습니다.

공동 사장님 추가

  1. 입력 칸에 추가할 회원의 아이디(닉네임)를 입력합니다.
  2. [+ 사장님 추가]를 누르면(엔터도 가능) "공동 사장님을 초대했습니다 — 운영자 승인 후 활성화됩니다" 안내가 뜹니다.
  3. 목록에 승인 대기 표시로 올라가고, 플랫폼 운영자가 승인하면 정식 공동 업주가 됩니다.
⚠ 닉네임이 존재하지 않는 회원이거나 조건이 맞지 않으면 초대가 실패하며 오류 안내가 뜹니다. 승인 전(대기 중)에는 아직 업주 권한이 없습니다.

40📢 팔로우 효과·팔로워 알림 보내기·매장 공지

팔로우가 주는 효과

손님이 매장 페이지 상단(매장명 오른쪽)의 팔로우 버튼을 누르면 우리 매장의 팔로워가 됩니다(로그인 필요). 버튼 옆 괄호 숫자가 현재 팔로워 수입니다. 팔로워가 되면 ①사장님이 직접 보내는 알림을 받고 ②새 대회(포스터) 등록 알림과 대회 하루 전(D-1) 리마인더가 자동으로 발송됩니다. 팔로워가 많을수록 홍보 도달 범위가 커지므로, 매장에서 손님에게 팔로우를 권하는 것이 가장 기본적인 마케팅입니다.

팔로워에게 알림 보내기 (마케팅 푸시)

내 매장 › 대시보드 하단의 「📢 팔로워에게 알림 보내기」 카드에서 보냅니다. 매장 관리 권한이 있는 계정에만 보입니다. 카드 우측 상단에 팔로워 수와 오늘 발송 횟수(예: 오늘 1/3)가 표시됩니다.

  1. 제목 입력 — 최대 60자 (예: 오늘 8시 GTD 500!)
  2. 내용 입력 — 최대 200자 (예: 마감 임박! 지금 예약하세요 🔥)
  3. 발송 버튼 클릭 → "팔로워 N명에게 푸시 알림을 보낼까요?" 확인창에서 확인
  4. 완료되면 "N명에게 발송했어요" 메시지와 함께 입력칸이 비워집니다
제한 항목내용
발송 한도하루 3회 — 소진 시 버튼이 "오늘 발송 한도 소진(3/3)"으로 잠깁니다
제목/내용제목 60자·내용 200자, 둘 다 비어 있으면 발송 불가
팔로워 0명버튼이 "아직 팔로워가 없어요"로 비활성화
권한발송 권한·한도는 서버에서도 한 번 더 검사 — 위반 시 에러
✅ 새 대회 등록·D-1 리마인더는 자동으로 나가므로, 하루 3회 수동 알림은 자동 알림과 겹치지 않는 내용(당일 긴급 공지, 좌석 마감 임박, 이벤트 추가 등)에 쓰는 것이 효율적입니다. 제목에 시간·보상(GTD)을 넣으면 열람률이 올라갑니다.
⚠ 발송은 취소·회수가 안 됩니다. 확인창에서 인원수를 보고 한 번 더 내용을 점검하세요. 한도 3회는 매일 초기화됩니다.

매장 공지

손님이 보는 매장 페이지 › 커뮤니티 탭 맨 위의 「매장 공지」 박스입니다. 열람은 누구나 가능하고, 작성·삭제는 우리 매장 업주와 총관리자만 할 수 있습니다.

  1. 커뮤니티 탭에서 + 공지 작성 클릭
  2. 내용 입력(최대 1,000자) 후 등록
  3. 삭제는 각 공지 우측 삭제 → 확인창에서 확인

공지가 하나도 없으면 손님에게는 이 박스 자체가 보이지 않습니다(업주에게만 작성 버튼이 보임). 등록된 공지에는 작성자명과 날짜가 함께 표시됩니다. 참고로 매장 관련 공지글은 포스터 탭의 「금일 포스터」를 펼쳤을 때도 최대 3건까지 함께 노출됩니다.

41⭐ 방문 후기(리뷰)와 사장님 답글

후기는 어디에 보이나

매장 페이지 › 매장 소개 탭 하단의 「방문 후기」 영역입니다. 별점 평균(소수 1자리)과 후기 건수가 후기 제목 옆에 표시되고, 매장 페이지 상단 매장명 옆에도 ⭐평점(건수)이 붙습니다. 이 평점은 매장 목록·일정 탐색 화면의 매장 카드에도 노출되므로 매장의 첫인상에 직접 영향을 줍니다.

손님이 후기를 쓸 수 있는 조건 (방문 인증제)

조건내용
로그인비로그인은 열람만 가능
체크인 기록우리 매장에서 QR 체크인을 1회 이상 한 회원만 작성 가능 — 가짜 후기 차단. 서버에서 강제되며, 체크인 없이 시도하면 "매장 QR 체크인 후에 후기를 쓸 수 있어요" 안내가 뜹니다
1인 1후기매장당 한 사람 한 개 — 다시 쓰면 기존 후기가 수정됩니다(버튼도 "내 후기 수정"으로 바뀜)
내용별점 1~5점(필수, 기본 5점) + 본문 최대 300자(선택)

후기 삭제는 작성자 본인 또는 총관리자만 할 수 있습니다(확인창 후 즉시 삭제). 업주는 남의 후기를 지울 수 없고, 대신 답글로 대응합니다.

사장님 답글 달기

우리 매장 업주(또는 총관리자)로 로그인하면 각 후기 아래에 답글 버튼이 보입니다.

  1. 후기 아래 🗨 답글 달기(이미 답글이 있으면 답글 수정) 클릭
  2. 답글 입력 — 최대 300자
  3. 답글 등록 클릭 → 후기 밑에 "사장님 답글" 배지와 함께 표시

답글을 빈 내용으로 등록하면 기존 답글이 삭제됩니다. 답글 권한은 서버에서도 매장 관리 권한을 확인하므로 다른 매장 후기에는 달 수 없습니다.

✨ AI 초안 버튼

답글 입력창 옆 ✨ AI 초안을 누르면 그 후기의 별점·내용을 읽고 정중한 존댓말 2~3문장 답글 초안을 자동으로 만들어 입력칸에 채워줍니다. 호평이면 감사+재방문 유도, 낮은 별점이면 사과+개선 약속 톤으로 작성됩니다.

✅ AI 초안은 어디까지나 초안입니다. 매장 상황에 맞게 한두 군데 고쳐서 등록하면 훨씬 진정성 있게 보입니다. 특히 낮은 별점 후기는 빠르고 구체적인 답글이 다른 손님들에게 좋은 인상을 남깁니다.
⚠ 후기 삭제는 되돌릴 수 없습니다. 또한 손님이 후기를 수정하면 기존 별점·내용이 새 내용으로 완전히 교체됩니다.

42🎨 매장 페이지 꾸미기 — 사진·소개·정보·탭 순서·내 매장 링크

누가 편집할 수 있나

승인된 업주가 본인 매장 페이지를 열면 상단에 「내 매장」 배지가 붙고, 아래 편집 버튼들이 나타납니다. 배지가 안 보이면 편집 권한이 없는 상태입니다.

사진(갤러리)·배경 관리

매장 페이지 › 상단 사진 영역 오른쪽 위 버튼으로 관리합니다.

  • 사진 추가 — 여러 장을 한 번에 선택해 갤러리에 추가. 사진이 2장 이상이면 3.5초마다 자동으로 넘어가고, 손님은 스와이프·좌우 화살표·하단 점으로 넘겨볼 수 있습니다(직접 넘기면 6초간 자동 넘김 일시정지)
  • 현재 사진 삭제 — 갤러리 사용 중일 때 왼쪽 위 ✕ 버튼으로 지금 보이는 사진 1장 삭제
  • 배경 업로드/변경 — 갤러리 사진이 하나도 없을 때만 보이는 단일 배경 방식. 5MB 이하 이미지만 가능
  • 사진이 전혀 없으면 매장 테마색 배경과 매장 이름 첫 글자가 대신 표시됩니다

탭 순서 바꾸기

내 매장 › 매장 꾸미기의 「매장 페이지 탭 순서」에서 매장 소개·순위·포스터·진행 예정·커뮤니티 5개 탭의 순서를 ↑↓ 버튼으로 조정합니다. 목록의 가장 위가 손님 화면의 가장 왼쪽(첫 탭)이 됩니다. 반드시 아래 탭 순서 저장을 눌러야 반영됩니다. 순위 탭에 보일 랭킹 보드 종류·1~3등 칭호·기준 점수는 여기가 아니라 「매장 랭킹」 탭에서 설정합니다.

내 매장 링크(주소 이름 정하기)

같은 화면의 「내 매장 링크」에서 공유 주소를 nuriholdem.com/s/원하는이름 으로 만들 수 있습니다.

  1. 이름 입력 — 영문 소문자·숫자·하이픈(-)만, 2~20자(대문자·한글은 자동 제거/변환)
  2. 중복 확인 클릭 — "✓ 사용할 수 있는 링크"가 떠야 저장 버튼이 활성화됩니다
  3. 저장 — 이후 매장 페이지의 링크 공유 버튼이 이 주소로 나갑니다

입력칸을 비우고 저장하면 기본 링크로 돌아가며, 커스텀 링크를 써도 기존 기본 링크는 계속 작동합니다. 손님이 링크 공유를 누르면 휴대폰 공유창이 뜨거나 링크가 자동 복사되고, 카톡에 붙이면 매장 미리보기 카드가 함께 나갑니다.

✅ 링크 이름은 매장명 영문(예: roti-arena)으로 정하면 손님이 기억하기 쉽고, 전단·명함·프로필에 그대로 인쇄해 쓰기 좋습니다.
⚠ 이미 다른 매장이 쓰는 이름은 등록되지 않습니다(중복 확인에서 ✕ 표시). 형식이 틀리면 "영문 소문자·숫자·하이픈으로 2~20자" 안내가 뜹니다.
