# 🎬 NURI HOLDEM 런칭 리허설 — 풀 퍼널 시나리오 (2026-06-12)

> 실계정 2개로 진행: **신규 손님 1명(새 폰)** + **사장님(업주/운영자, PC)**.
> 순서대로 따라가면 가입→체크인→미션→보상→후기→예약→장부→순위→랭킹까지 전부 한 바퀴 돕니다.
> 각 단계의 ✅가 전부 통과하면 런칭 준비 완료입니다.

---

## 0단계. 준비 (PC·사장님)
- [ ] 매장 QR 출력물 준비: 내 매장 → 대시보드 → **입장 QR** (회원가입 QR도 옆에 있음)
- [ ] 오늘 포스터 1개 등록: 내 매장 → 포스터·예약 → **+ 새 게임** (지난 포스터 불러오기로 30초)
- [ ] 관리자 설정 → 게시물 관리 → **주간 미션 관리**에서 이벤트 미션 1개 추가(예: "오픈 기념 체크인 1회 +50점")

## 1단계. 가입 (폰·손님)
- [ ] nuriholdem.com 접속 → 회원가입(이메일) → 본인인증(PortOne)
- ✅ 닉네임 중복 체크 동작 / 만 19세 게이트 / 가입 직후 로그인 상태
- [ ] PWA 설치 배너 → 홈 화면에 추가

## 2단계. 탐색 + 예약 (폰·손님)
- [ ] 일정 탐색에서 오늘 포스터 확인 (⭐별점은 후기 쌓이면 매장명 옆 자동 표시)
- [ ] 포스터 탭 → **두 손가락 핀치줌**으로 확대 → 닫기
- [ ] **참가 예약** → 예약 완료 → 헤더 알림/예약 뱃지 확인
- ✅ PC 게임관리 행에 **예약 1** 칩 실시간 반영 (사장님 화면)

## 3단계. 체크인 + 미션 (폰·손님, 매장에서)
- [ ] 매장 QR 스캔 → 체크인 → "+3점 도장" 토스트
- ✅ 커뮤니티 → 랭킹 → **미션** 보드: 체크인 미션 진행도 1/2 (커스텀 미션도 1/1)
- [ ] 달성된 미션 **🎁 받기** → 점수 지급 토스트 → 활동 점수 상승 확인
- ✅ 랭킹 → 주간 리그에 본인 등장(체크인 ×3점)

## 4단계. 후기 (폰·손님)
- [ ] 매장 페이지 → 매장 소개 탭 → **후기 쓰기** 버튼 노출(체크인했으므로)
- [ ] 별점 5 + 한 줄 작성 → 등록
- ✅ 일정탐색 카드 매장명 옆 **⭐5.0(1)** 표시 / 커뮤니티 매장 카드에도 표시
- ✅ (교차검증) 체크인 안 한 다른 계정에선 "체크인 회원만" 안내만 보임

## 5단계. 장부 + 클락 (PC·사장님)
- [ ] 내 매장 → 포스터·예약 → 오늘 포스터 행 **장부 +** 클릭
- ✅ 장부 시작 설정에 게임명·바인 단가·포스터 연결 자동 입력됨
- [ ] 담당 직원 선택 → **장부 입장** → "클락도 같이 켤까요?" → **확인**
- ✅ 클락이 장부 날짜로 연동돼 열림 / 담당 직원 계정에 "📒 장부 시작" 알림 도착
- [ ] 손님 닉네임으로 유저 추가(가입계정 연동 제안 확인) → 바인 1개 입력
- ✅ 게임관리 행 미니칩 **바인 1 · 매출 N만** 갱신

## 6단계. 마감 + 순위 (PC·사장님)
- [ ] 장부 마감(메모 입력) → 게임관리로 이동
- ✅ 포스터 행에 **🟠 순위 미입력** 뱃지 노출
- [ ] 뱃지 클릭 → 순위 입력 화면(해당 날짜) → 손님 닉네임 1위 입력 + 이용권 1개 지급 → 저장
- ✅ 뱃지가 **마감 · 순위 ✓** 칩으로 변경

## 7단계. 보상 확인 (폰·손님)
- [ ] 헤더 🎟 지갑: **이용권 1개** 수신 확인
- [ ] 개인 대시보드: 입상 기록 1위 표시(입상 2회부터 추이 그래프 자동)
- ✅ 랭킹 → 머니인 보드 + 지난 대회 아카이브에 👑 닉네임 노출
- ✅ 미션 "머니인 1회" 달성 → 🎁 +30점 수령

## 8단계. 마무리 점검 (둘 다)
- [ ] 모바일: 스크롤 시 헤더 축소 / 게시판 글 1개 작성(미션 +10) / 예약 내역 좌스와이프 취소 시험
- [ ] PC: 관리자 설정 → 오류 로그 **0건** 확인
- [ ] 카카오톡에 공유 링크(?s=) 보내서 미리보기 확인
